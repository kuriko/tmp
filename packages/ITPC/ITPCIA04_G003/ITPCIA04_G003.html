アプリを起動してみよう

今回は macOS でアプリを起動し、終了する手順を学びましょう。
macOS に標準で入っているアプリを使います。

  • 計算機（電卓）を起動する
  • 計算機を終了する
  • 株価アプリを起動する
  • 株価アプリを起動させたまま、計算機を起動する
  • アプリを切り替える
  • 株価アプリと計算機を終了する
  • アプリを起動する別の方法
  • 強制終了

計算機（電卓）を起動する

ここは非常に重要な内容です。きちんと操作を理解しましょう！

今回は計算機アプリを起動してみましょう。起動方法は他のアプリでも一緒ですが、ここでは皆さんの mac パソコンに必ず入っている電卓を使います。

まず、画面下部の Dock（ドック）と呼ばれるエリアにある、 Launchpad（ローンチパッド）をクリックして開きます。

Dock から Launchpadを起動

Launchpad はインストールされているアプリの一覧を表示します。1 枚の画面に表示しきれない分は左右にスクロールすることができます。
電卓機能を持つ 計算機 というアプリを探し、クリックしましょう。

Launchpad

計算機を終了する

ここは非常に重要な内容です。きちんと操作を理解しましょう！

電卓アプリを終了するには、画面上部にあるメニューバー の 計算機 をクリックし、計算機を終了 をクリックします。

計算機アプリの終了

もし、メニューバーが 計算機 となっていない場合、他のアプリを操作する状態になっています。
そのときは Dock で「計算機」アプリのアイコンをクリックしましょう。


アイコンをクリックして状態を切り替えている様子
コラム:　Windows との違い

Windows を使ったことがある人は、ウィンドウを閉じる ＝ アプリの終了 と覚えていると思います。
しかし、macOS ではウィンドウを閉じるだけではアプリは終了しません。いわゆる「常駐」のような、待機中の状態に入ります。
Mac では一旦閉じてまたすぐに使い始める時は「ウィンドウを閉じる」、明確に終了する場合は「終了」というように使い分けるとよいでしょう。

アプリを切り替える

ここは非常に重要な内容です。きちんと操作を理解しましょう！

起動したアプリのアイコンが画面の右下に並んでいると思います。 先程も出てきましたが、この部分をDock（ドック）といいます。
Dock では、起動中のアプリのアイコンの下に黒い丸（●）が表示されています。

株価アプリと計算機アプリのアイコンが並んでいる

Dock で株価と計算機のアイコンをそれぞれクリックしてみてください。
操作する対象となるアプリが切り替わるのではないかと思います。

このように、Dockのアイコンをクリックすることで、アプリを切り替えることができます。

また、両方のアプリが並んで表示されているときは、アプリのウィンドウをクリックすることで切り替えることもできます。


ウィンドウのクリックで切り替えている様子

株価アプリと計算機を終了する

ここは非常に重要な内容です。きちんと操作を理解しましょう！

計算機は先程と同じように、画面上部にあるメニューバー の 計算機 をクリックし、計算機を終了 をクリックしましょう。

株価アプリも同じ方法で終了できますが、今回は別の方法で終了してみましょう。
Dockにある株価アプリのアイコンを右クリックしてください。続いて、終了 をクリックしましょう。

株価アプリを Dock から終了させる

このように、Dock からもアプリを終了させることができます。

アプリを起動する別の方法

ここは重要な内容です。なるべく操作を理解しましょう。

今回は Launchpad から起動しましたが、他の起動方法も知っておきましょう。

Dock から起動する

画面の下部にあるDockには、起動していないアプリも並んでいるはずです。
実は、よく使うアプリをここに登録することもできます。

Dock
Dock にアプリを登録する

アプリを起動した状態でアイコンを右クリックし、オプション をクリックします。続いて Dock に追加 をクリックする（チェックを入れる）ことで自分でアプリを登録することができます。
なお、起動しているアプリはアイコンの下に黒い丸が表示されます。

Dock に追加する
Dock からアプリを解除する

もし必要なくなったら、同様にアイコンを右クリックして オプション をクリックし、続いて Dock に追加 をクリックする（チェックを外す）ことで解除できます。
この操作をしてもアプリ自体が消えるわけではないのでご安心ください。

Dock から消す

Spotlight 検索を使う

ここは応用的な内容です。余裕のある人は操作してみましょう。

先程は Launchpad にあるアプリを探しましたが、アプリが増えてくると探すのが大変になります。
そういうときは検索機能を使うことができます。

画面右上にある虫眼鏡マークをクリックすると、Spotlight（スポットライト）検索が表示されます。

Spotlight

起動した Spotlight 検索に「計算機」のようにアプリの名前を入力し、候補からダブルクリックすることでアプリを起動できます。
なお、キーボードの操作についてはこのあと学んでいくので、現時点でよくわからなくても心配ありません。

強制終了

ここは重要な内容です。操作を覚えておきましょう！

ごくまれに、アプリの動作がおかしくなり、終了しようとしても操作を受け付けなくなる場合があります。
そういう時は macOS が備えている強制終了機能を使いましょう。

なお、強制終了を使うとアプリの保存していないデータが破棄されることがあるので、本当に必要なときだけ使うようにしましょう（ただし、強制終了によってアプリやパソコンが壊れるようなことはほぼ無いので、過度に不安になる必要はありません）。
今回は「計算機」アプリを強制終了してみます。このアプリは特にファイルを保存することがないので、強制終了の練習に使って大丈夫です。

まず、Launchpad から 計算機アプリ を起動します。

実際にはきちんと動いていますが、計算機アプリが動かなくなってしまった……と仮定します。
画面左上にあるリンゴマークをクリックし、強制終了をクリックします。

アプリの強制終了メニュー

アプリケーションの強制終了 というウィンドウが出てきたら、計算機をクリックします。 続いて、強制終了をクリックします。

計算機を強制終了する

「"計算機"を強制終了してもよろしいですか？」との表示が出るので、強制終了をクリックします。

確認画面

これで強制終了できました。普段から使う機能ではありませんが、万が一に備えて手順を覚えておくとよいでしょう。
