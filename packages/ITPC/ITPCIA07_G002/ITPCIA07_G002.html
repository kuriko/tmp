ネットでキーボードの入力練習をしよう

パソコンでのキーボード入力（タイピング）を習得するためには、実際にたくさん入力することが一番です。
今回は楽しみながらタイピングに慣れられる無料のゲーム「寿司打」（すしだ）をご紹介します。

寿司打はMyaさんによって開発・提供されています。

  • 寿司打にアクセスしよう
  • 寿司打をプレイしよう
  • その他のタイピングゲーム
  • LINE をパソコンから使ってみる

寿司打にアクセスしよう

「寿司打」と Google 検索すると一番上に出てくるはずです。
「【寿司打】WebGL版 - Neutral タイピング」をクリックしましょう。

寿司打のGoogle検索結果

また、以下の URL を Chrome のアドレスバーへ入力してもアクセスできます。
http://typingx0.net/sushida/
（x の後の 0 はアルファベットの O（オー）ではなく、数字の 0 （ゼロ）です。）

寿司打をプレイしよう

寿司打にアクセスすると、以下のような画面になります。

寿司打のトップ画面

寿司打をPlay ボタンをクリックすると、ゲームのページに移動します。

寿司打のプレイ開始ボタン

少しすると以下のような画面が表示されます。
画面の真ん中にある スタート をクリックしましょう。

寿司打のスタート画面

難易度とコース選択

スタートすると、難易度とコースを選択する画面が表示されます。
なお、寿司打で表示されている「3000円」などの金額はゲーム内でのスコアであり、実際にお金を支払うことは全くないのでご安心ください。

寿司打の難易度を選ぶ画面

初心者の場合、難易度は以下のどちらかにしておくと良いでしょう。

  • 練習：時間制限がゆっくりになります。最初はこれを選んでみましょう。
  • 普通：通常のモードです。「練習」に慣れたらこちらに切り替えてみましょう。

「コース」はゲーム中で入力する単語の文字数です。
最初は「お手軽」からスタートしてみると良いでしょう。

  • お手軽 3,000円コース（2～7文字）
  • お勧め 5,000円コース（5～10文字）
  • 高級 10,000円コース（9～14文字）

コースをどれかクリックするとゲームが開始されます。

ゲーム開始

以下の画面で スペース キー か Enter キーを押すとゲームが始まります。

寿司打の開始画面

表示された日本語に対して、ローマ字を入力します。例えば、部下（ぶか）と表示されたら buka と入力します。
正確に入力できると、次の単語に切り替わります（変換やEnterキーは必要ありません）。制限時間内になるべく多くの単語を入力していくゲームです。

寿司打のプレイ中画面

寿司打をプレイしている様子

ゲーム終了

ゲームが終了すると結果が表示されます。「お得でした」と表示されている場合はそのコースをクリアできています。
ただ、クリアしたかどうかだけではなく、「正しく打ったキーの数」「平均キータイプ数」「ミスタイプ数」にもきちんと注目しましょう。

寿司打のトップ画面
  • 正しく打ったキーの数: 入力したキーのうち、正しく入力できたものの数です。
  • 平均キータイプ数: 1秒間に入力したキーの数です。これが3～4回であれば一般的には十分な速度といえるでしょう。
  • ミスタイプ数: 入力ミスをした回数です。なるべく減らすように心がけましょう。

クリアした場合、自分のスコアがランキングでどれくらいの位置かを確認できます。「ランキングを表示」をクリックして確認してみましょう。
ランキングを上げることを目標に頑張るのもよいでしょう。

寿司打のランキング表示画面

その他のタイピングゲーム

今回は寿司打を紹介しましたが、タイピングゲームは他にもたくさんあります。
色々なサイトで試してみるのもよいでしょう。

Popタイピング

URL: http://typingx0.net/pop/

寿司打と同じサイトで公開されているゲームです。ポップコーンが題材となっています。
表示された単語を入力していくシンプルなものです。

e-typing（イー・タイピング）

URL: https://www.e-typing.ne.jp/
「e-typing」と Google 検索すると一番上に出てくるはずです。

ジャンル別の単語や短文でタイピングを練習できます。
それぞれ分かりやすい「レベル」が表示されるほか、苦手なキーも表示されるので、次の目標を立てやすいしょう。

P検 無料タイピング練習

URL: https://www.pken.com/tool/typing.html

ICTプロフィシエンシー検定試験（旧称 : パソコン検定）、略称「P検」のサイトで提供されているタイピング練習サイトです。
ホームポジションの習得や英語での入力練習も可能です。

LINE をパソコンから使ってみる

タイピングゲームではありませんが、普段スマホで入力している LINE をパソコンから使ってみるのもキーボード（タイピング）に慣れる一つの手です。
