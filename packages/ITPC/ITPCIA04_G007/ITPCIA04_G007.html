Finder の機能

Finder では、表示するスタイルの変更やファイル・フォルダの並べ替えができます。

表示スタイル

Finder の上部にあるボタンから、アイコンの表示を変えることができます。

  • アイコン表示 : ファイルを大きなアイコンで表示します。
  • リスト表示 : ファイルを詳細な情報付きのリストで表示します。
  • カラム表示 : 階層構造になったフォルダを一括で表示できます。
  • ギャラリー表示 : ファイルの内容を大きくプレビューします。画像が多く含まれたフォルダに適しています。

自分が使いやすい表示に切り替えて使いましょう。

グループ分け

同じく Finder の上部にある、四角が6つ並んだボタンをクリックしてみましょう。 これは グループ分け のボタンで、名前や更新日時などによってグループ分けをすることができます。
この機能はファイルの数が増えてきたときに、目的のファイルを見つけるために非常に役立ちます。

グループ分け

表示順序

グループ分けだけでなく、並び替えることもできます。

画面最上部にある Finder のメニューから 表示 をクリックし、 表示順序 からファイルを並び替えることができます。
この機能も目的のファイルを見つけるために非常に役立ちます。

ファイルとアプリの関連付け

実は、テキストエディットで作成したファイル（リッチテキストファイル）はテキストエディット専用のファイルというわけではありません。
テキストエディットよりも高機能なテキストファイル編集用のアプリもあり、それらで開くこともできます。

また、画像ファイルなどでも同様のことが言えます。
画像ファイルは画像を見るためのアプリでも、画像を編集するためのアプリでも開けます。

//TODO 説明イラスト: 1つのファイルは複数のアプリケーションで開ける

このように、パソコンにおいてファイルは「特定のアプリ専用」ではなく、他のアプリで開くことができるものがあります。

しかし、開くたびにアプリを指定しているとちょっと面倒です。
そのため、「Finder からファイルをダブルクリックした時には、このファイルの種類は基本的にこのアプリで開く」という 関連付け が設定されています。

関連付けられたアプリ以外でファイルを開きたい場合、Finder でファイルを右クリックし、「このアプリケーションで開く」というメニューをクリックすることで別のアプリを選択できます。

複数のファイルをまとめて操作する

Finder では、キーボードの command キーを押しながら、マウスでファイルやフォルダをクリックすることで複数のファイルやフォルダを選択できます。

複数のファイルやフォルダを選択

選択を解除する場合は何もない部分をクリックしましょう。

選択を解除

また、連続して並んでいるファイルやフォルダを選択するときは、まず始まりのファイルをクリックします。続いて、Shift キーを押しながら終わりのファイルをクリックします。
（なお、この機能は「アイコン表示」では使えません）

連続したファイルやフォルダを選択

このように、複数のファイルやフォルダが選ばれた状態で右クリックすると、一気に移動や削除の操作を行うことができます。

ファイルやフォルダの容量（サイズ）の確認

ファイルやフォルダを右クリックし、情報を見るをクリックします。

情報を見る

情報ウィンドウの「サイズ」の欄で確認できます。

サイズ

なお、容量が大きいフォルダを開いた場合はファイルサイズを集計する時間が掛かるため、最初は「計算中」と表示されます。

ストレージ容量の確認

スマホなどと同様に、パソコンもストレージの容量には限度があります。いまどれくらい使っているのかを定期的にチェックしましょう。

メニューバーの左上のりんごマークから「この Mac について」を選択します。出てくるウィンドウの「ストレージ」タブをクリックします。すると、PC 本体のハードディスクの空き容量を、おおまかな内訳とともに確認することができます。

容量の大きいファイルは 圧縮（あっしゅく）することができます。
Web サイトに掲載したり、人にメールなどで送ったりする場合に便利です。

Finder は ZIP（ジップ）という形式での圧縮に対応しています。

ファイルやフォルダを圧縮する

Finder で 書類 を開き、勉強 フォルダを開きましょう。続いて、さらにその中にある パソコン フォルダを開きましょう。

前回の「ファイルやフォルダ操作の基本を学ぼう」で作成した、「ファイルの練習」というファイルを 右クリック し、"ファイルの練習"を圧縮 をクリックします。

ファイルの圧縮

すると、「ファイルの練習.zip」という名前でチャックの付いたアイコンが登場したはずです。これが ZIPファイル（ZIP形式ファイル）です。

ZIP ファイルを展開する

続いて、ZIP ファイルを 展開（てんかい） してみましょう。展開とは、圧縮されたものを普通のファイル・フォルダに戻すことです。冷凍品に例えて解凍（かいとう）と呼ばれることもあります。

Finder では、ZIP ファイルを ダブルクリック すると展開が始まります。
今回は先ほど作成した 勉強.zip を展開してみましょう。勉強.zip をダブルクリックしてください。
既に書類フォルダに「勉強」というフォルダがあるので、名前が被らないように「勉強 2」というフォルダが作成されました。

フォルダが展開された

ここに先程圧縮したファイルやフォルダが入っています。

展開されたフォルダの中身

今回は自分で圧縮したフォルダを展開しましたが、実際には人から送られてきたファイルやダウンロードしたファイルに対してもこの展開操作をしていくことになります。

圧縮の注意事項

ファイルを圧縮には限度があるため、ファイルの種類によっては圧縮してもファイルサイズが小さくならないものもあります。
また、他の OS (Windows など) で圧縮されたファイルを開くと、その OS 独自のファイルが混入したり、ファイル名が文字化けすることがあります。

コラム: ファイルサイズを小さくするだけじゃない

ZIP 圧縮の主要な目的はファイルサイズを小さくすることですが、複数のファイルを 1 つのファイルにまとめることができるというメリットもあります。

拡張子

余裕がある人はやってみましょう。これは高度な内容なので、難しいと思ったら読み飛ばしてかまいません。
また、今後プログラミングをやってみたいという人は知っておいて損はない内容です。

拡張子（かくちょうし）とは、ファイルの種類を区別するためにファイルの末尾に加えられる文字のことです。
たとえば先程作成した ZIP ファイルの末尾には .zip という文字がついていましたが、これも拡張子です。

拡張子

macOS の標準設定では、殆どのファイルの拡張子を非表示にするようになっています。これは誤って拡張子を編集してファイルが認識されなくなるトラブルを防ぐためです。
拡張子を表示したい場合、Finder のメニューから 環境設定 をクリックし、 詳細 から 「すべてのファイル名拡張子を表示」にチェックを入れてください。

環境設定
詳細

なお、拡張子はあくまで「区別するため」の文字です。拡張子を変えただけでファイルの形式は変わりません。
たとえば Word というアプリのファイル（拡張子 .docx）を PDF という形式を表す .pdf に変えたからといって、PDF ファイルに変換されるわけではありません。

ファイルパス・フォルダパス

余裕がある人はやってみましょう。これは高度な内容なので、難しいと思ったら読み飛ばしてかまいません。
また、今後プログラミングをやってみたいという人は知っておいて損はない内容です。

ファイルやフォルダの位置を文字列で示したものを ファイルパス や フォルダパス といいます。パス（path）は「小道」という意味で、そのフォルダやファイルにたどり着くための道をイメージするとよいでしょう。

Mac でフォルダのパスは /Users/your_name/Documents/勉強/パソコン のように表されます。ファイルのパスは /Users/your_name/Documents/勉強/パソコン/練習.txt のように、フォルダのパスの後にファイル名が加えられたものです。

Finder で、ファイルやフォルダを右クリックして option キーを押したままで "ファイル名・フォルダ名"のパスをコピー をクリックするとパスをコピーできます。

ファイル名・フォルダ名のパスをコピー

また、Finder のメニューにある 移動 から フォルダへ移動 をクリックし、パスを直接入力すると、そのフォルダを開くこともできます。
