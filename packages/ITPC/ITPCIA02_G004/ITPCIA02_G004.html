アプリを起動してみよう

今回は Windows でアプリを起動し、終了する手順を学びましょう。
Windows に標準で入っているアプリを使います。

  • 今回の目標
  • 電卓を起動する
  • 電卓を終了する
  • メモ帳を起動する
  • メモ帳を起動させたまま、電卓を起動する
  • アプリを切り替える
  • メモ帳と電卓を終了する
  • アプリを強制終了する
  • まとめ

今回の目標

  • パソコンにおけるアプリの起動と終了手順を学ぶ
  • 複数のアプリを同時に使う方法を知る
  • いざという時に備えて、アプリを強制終了する方法を学ぶ

電卓を起動する

ここは非常に重要な内容です。きちんと操作を理解しましょう！

今回は電卓アプリを起動してみましょう。起動方法は他のアプリでも一緒ですが、ここでは皆さんの Windows パソコンに必ず入っている電卓を使います。

まず、画面左下のスタートメニューをクリックして開きます。
ここに、インストールされているアプリの一覧が表示されています。

スタートメニューを開く

続いて、スクロールして 電卓 をクリックします。

電卓 をクリック

電卓アプリが起動しました。

電卓アプリが起動した

電卓を終了する

ここは非常に重要な内容です。きちんと操作を理解しましょう！

電卓アプリを終了するには、ウィンドウ右上にある閉じるボタンをクリックします。
閉じるボタンはマウスでポイントすると赤色になります（アプリによっては色が変わらないものもあります）。

電卓アプリを終了する

メモ帳を起動する

ここは非常に重要な内容です。きちんと操作を理解しましょう！

続いて、メモ帳アプリを起動してみましょう。

まず、画面左下のスタートメニューをクリックして開きます。

スタートメニューを開く

続いて、スクロールして Windows アクセサリ をクリックします。
その中に入っている メモ帳 をクリックします。

メモ帳 をクリック

メモ帳アプリが起動しました。

メモ帳が起動した

メモ帳を起動させたまま、電卓を起動する

ここは非常に重要な内容です。きちんと操作を理解しましょう！

パソコンでは、複数のアプリを同時に起動して並べて使うことができます。

先ほどと同じ手順で、スタートメニューから電卓を起動してみましょう。

メモ帳と電卓が同時に起動している

メモ帳と電卓が並んだのではないでしょうか。

アプリを切り替える

ここは非常に重要な内容です。きちんと操作を理解しましょう！

起動したアプリのアイコンが画面の下に並んでいると思います。 この部分をタスクバーといいます。

タスクバー

タスクバーでメモ帳と電卓のアイコンをそれぞれクリックしてみてください。
若干ですが色が変わると思います。

アプリを切り替える様子

色が濃くなっている方が現在使用しているアプリです。
このようにタスクバーのアイコンをクリックすることで、アプリを切り替えられます。

タスクバーで使用するアプリを切り替える

また、両方のアプリが並んで表示されているときは、アプリのウィンドウをクリックすることで切り替えることもできます。

ウィンドウで使用するアプリを切り替える

メモ帳と電卓を終了する

ここは非常に重要な内容です。きちんと操作を理解しましょう！

電卓は先ほどと同じように、閉じる ボタンをクリックします。

メモ帳も閉じるボタンがあるのですが、今回は別の方法で終了してみましょう。
メモ帳のファイル(F) をクリックしてください。

続いて、メモ帳の終了(X) をクリックしましょう。

このように、ファイル メニューから終了できるアプリもあります。
また、アプリによってはファイルメニューからでないと完全に終了できないものもあるので、アプリの説明などを読んでから使い分けましょう。

アプリを起動する別の方法

今回はスタートメニューから起動しましたが、他の起動方法も知っておきましょう。

タスクバーから起動する

画面の下部にあるタスクバーには、起動していないアプリも並んでいるはずです。
実は、よく使うアプリをここに登録することもできます。

タスクバー

アプリを起動した状態でアイコンを右クリックし、タスクバーにピン留めする をクリックすることでアプリを登録できます。

タスクバーにピン留めする

もし必要なくなったら、同様にアイコンを右クリックして タスクバーからピン留めを外す をクリックすることで解除できます。

タスクバーからピン留めを外す

デスクトップのショートカットから起動する

パソコンを起動した直後から表示されている画面をデスクトップといいます。スマホのホーム画面のようなものです。
アプリによっては、インストール時にデスクトップにアイコンを作成するものがあります。ここのアイコンをダブルクリックすることで起動できます。

デスクトップのショートカットから起動する

デスクトップにあるアプリのアイコンをショートカットといいます。ショートカットは「近道」という意味の英単語です。

スタートメニューで検索する

ここは応用的な内容です。余裕のある人は操作してみましょう。

先ほどはスタートメニューにあるアプリを探しましたが、アプリが増えてくると探すのが大変になります。
そういうときは検索機能を使うことができます。

スタートメニューを開いた状態で、キーボードからアプリの名前を入力してみましょう。
すると、当てはまるアプリが表示されるので、クリックすると起動できます。

スタートメニューで検索する

なお、キーボードの操作についてはこのあと学びます。現時点でよくわからなくても心配ありません。

アプリを強制終了する

ごくまれに、アプリの動作がおかしくなり、終了しようとしても操作を受け付けなくなることがあります。
そういう時は Windows の強制終了機能を使いましょう。

なお、強制終了を使うとアプリの保存していないデータが破棄されることがあるので、本当に必要なときだけ使うようにしましょう（ただし、強制終了によってアプリやパソコンが壊れるようなことはほぼ無いので、それほど心配しなくてよいです）。
今回は「電卓」アプリを強制終了してみます。このアプリは特にファイルを保存することがないので、強制終了の練習に使って大丈夫です。

まず、スタートメニュー から 電卓アプリ を起動します。

実際にはきちんと動いていますが、電卓アプリが動かなくなってしまった……と仮定します。
画面下部にあるタスクバーの何もないところを右クリックし、タスクマネージャーをクリックします。

「タスク マネージャー」をクリック

タスクマネージャーに表示される 電卓 を右クリックし、タスクの終了をクリックします。

「タスクの終了」をクリック

これで強制終了できました。普段から使う機能ではありませんが、万が一に備えて手順を覚えておくとよいでしょう。

まとめ

  • 画面左下のスタートメニューからアプリを起動できる。
  • アプリの右上にある閉じるボタンをクリックすると、アプリを終了できる。
  • アプリは同時に複数起動でき、並べて表示できる。
  • 同時に起動しているアプリは、画面下部のタスクバーから切り替えできる。
  • アプリが動かなくなったときは、タスクマネージャーから強制終了できる。
