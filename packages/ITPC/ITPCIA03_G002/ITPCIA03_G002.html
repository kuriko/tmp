電源のオン・オフと Mac のスペック確認

Mac での電源の操作を学びましょう。また、起動したあとに自分が使っている Mac のスペック（性能・バージョン）を確認してみましょう。

その前に、電源のアイコンを覚えておきましょう。
これは電子機器における 電源 のアイコンです。スマートフォンやゲーム機などでも目にした事があるのではないでしょうか。
一見するとリンゴのようにも見えますが、デジタルの 0（オフ） と 1（オン）を意味しているといわれています。

起動方法（スマホとの違い）

MacBook シリーズ（MacBook, MacBook Air, MacBook Pro）では、Mac の蓋（ふた）を開くと電源がオンになります。
もし、既に蓋（ふた）を開いている状態で電源がオフになっており、改めて電源を入れたい場合は電源ボタンを操作してください。

なお、電源ボタンを操作する場合は長押しする必要はありません。1 回カチッと押せば大丈夫です。

電源ボタンの場所

MacBook シリーズの一部では、キーボード右上にある指紋センサーが電源ボタンになっています。
電源マークがないと思ったら右上を確認してみましょう。

指紋センサーが電源ボタンになっている

デスクトップ型（iMac）の場合、画面の裏側に電源ボタンが配置されています。

バージョン

確認してほしいものの 1 つは バージョン です。
バージョンといっても細かい数字ではなく、macOS に付けられた Big Sur（ビッグサー） や Catalina（カタリナ）といった名前が分かっていれば大丈夫です。

macOS のバージョン

macOS 10.13 High Sierra（ハイ・シエラ）
macOS 10.14 Mojave（モハベ）
macOS 10.15 Catalina（カタリナ）
macOS 11 Big Sur（ビッグサー）

Catalina 以前（Sierra、High Sierra、Mojave、Catalina）と Big Sur では画面の見た目などが異なっています。
そのため、この教材では Catalina 以前 と Big Sur で解説を分けている場所があります。自分が使っているバージョンに合ったものを読んでください。

プロセッサ・チップ

もう 1 つは プロセッサ や チップ です。
詳細な型番を確認する必要はないのですが、ここに Intel（インテル）と書かれているか、Apple M1（アップル・エムワン）と書かれているかを確認しておいてください。

Intel と書かれているものは Intel 製 CPU を搭載した 従来型の Mac 、Apple M1 と書かれているものは Apple 製 CPU を搭載した 新型の Mac です。
どちらも操作は変わりません。ただし、Apple M1 搭載モデルは最新版のため、まだ対応していないアプリもあります。

自分がどちらの Mac を使っているかを知っておくことが重要です。

バッテリー残量の確認

ノートパソコンの場合、バッテリーの残量を確認しましょう。もしバッテリー残量が低下してきたら、0% になってしまう前に早めに充電をしましょう。
Mac では画面右上のメニューバーに表示されます。

初期設定ではバッテリー残量の数字（％）は表示されません。表示させるためには、メニューバーの電池マークのアイコンをクリックし、「割合（%）を表示」を選択します。

「割合（%）を表示」をクリック

これで、メニューバーに数字（％）が表示されるようになりました。

パソコンの終了（スマホとの違い）

スマホでは電源ボタンを押してスリープさせたり、電源オフにしたりしますが、パソコンでは画面上で操作して電源を切ります。
画面左上にリンゴマークが描かれているアップルメニューをクリックします。

このメニューには スリープ、再起動、システム終了 の 3 つが並んでいます。それぞれの意味を知っておきましょう。

スリープにする

スリープとは、パソコンで作業中の内容（開いているアプリなど）を保持したまま、PC の動作を止める機能です。
なお、スリープ中は完全に電源が切れているわけではなく、ほぼ電力を使わない状態で電源が入っています。

MacBook シリーズの場合、蓋（ふた）を閉じることでもスリープになります。

システム終了

パソコンを終了するときの最も基本的な操作です。

これをクリックしてすぐに電源が切れるのではなく、少しだけ終了に関する処理が行われた後に電源が切れます。
電源が切れていない間にノートパソコンを閉じたり、収納したりするのは避けましょう。

再起動する

再起動とは、シャットダウンにしたあとにもう 1 回起動することです。
新しいアプリをインストールした後に内容を反映させる時や、パソコンの動作が不安定な時にリセットする目的などで使われます。

シャットダウンと違うのは、電源が切れたあとに自動的に再び電源がオンになることです。

PC の強制終了

PC の動作がおかしくなってしまった場合、強制終了しなければならないときがあります。いざというときに備え、手順を覚えておきましょう（いま試す必要はありません）。

まずは、control キー と option キーと command キーと 電源ボタン を同時に押します。
4つのキー・ボタンを同時押しすることになるので、ちょっと大変です。
上手くいくと、Mac の終了に関するメニューが現れるので、画面の指示に従って操作して終了してください。もし、保存してないデータがある場合は保存することもできます。

もし、上記の手順で終了に関するメニューが現れない場合は control キー と command キーと 電源ボタン を同時に押します。
この手順の場合、保存していないデータがあっても無視されます。

それでも動かない場合

この手順による強制終了はデータ破損など、トラブルの原因になることもあります。PC が操作を受け付けなくなるなど、やむを得ない場合のみに使用してください。

キーボードやマウスの操作をしても全く動かない場合、Mac の電源ボタンを長押しします。
電源が切れるまで長押しし続けてください。

まとめ

  • Mac は蓋を開くか、電源ボタンをカチッと 1 回押すと起動できる。長押しする必要はない。
  • バッテリー残量は画面の右上で確認できる。
  • 電源を切るときは左上アップルメニューからシステム終了をクリックする。
  • いざというときに備えて、強制終了の手段を知っておく。
