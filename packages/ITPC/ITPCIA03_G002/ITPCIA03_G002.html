電源のオン・オフ

Windows パソコンでの電源の操作やバッテリー残量の確認方法を学びましょう。

その前に、電源のアイコンを覚えておきましょう。
これは電子機器における 電源 のアイコンです。スマートフォンやゲーム機などでも目にした事があるのではないでしょうか。
一見するとリンゴのようにも見えますが、デジタルの 0（オフ） と 1（オン）を意味しているといわれています。

起動方法（スマホとの違い）

「電源ボタンを押すと起動できる」という点ではスマホとほとんど変わりませんが、ノートパソコンでは蓋（ふた）を開くことで自動的に電源がオンになる製品もあります。

なお、電源ボタンを操作する場合は長押しする必要はありません。1 回カチッと押せば大丈夫です。

電源ボタンの場所

電源ボタンの例

電源ボタンの位置は製品によって異なるので、お使いのパソコンの説明書をご覧ください。
パナソニック製の Let's note（レッツノート） シリーズなど、ボタンではなくスライドするタイプのスイッチを搭載した PC もあります。

ログインする

パスワードを設定していたり、パソコンに複数のユーザを登録している場合、起動中にログイン処理を行う必要があります。
時計が表示されたら Enter と書かれたキーを押し、パスワードを入力して再び Enter キーを押します。

バッテリー残量の確認

ノートパソコンの場合、バッテリーの残量を確認しましょう。バッテリー残量が低下してきたら、0% になってしまう前に早めに充電をしましょう。

画面の右下にバッテリーのアイコンがあります。ここにマウスの矢印（ポインタ）を合わせましょう。クリックする必要はありません。

バッテリーアイコンを探す

矢印を合わせると、残り利用可能時間とバッテリー残量が表示されます。また、コンセントに接続している場合は「電源に接続中」と表示されます。
この状態でバッテリーのアイコンをクリックしてみましょう。

バッテリーのアイコンに表示される残量

すると、より大きな表示になります。また、この画面からバッテリーに関する設定を変更することもできます。
キーボード左上にある Esc（エスケープ）というキーを押すと、この表示は消えます。

バッテリーの詳細表示

なお、残り時間はあくまで予測値なので、パソコンの使用状況などによって変動します。

パソコンの終了（スマホとの違い）

基本的にスマホでは電源ボタンを押して終了（電源オフ）しますが、パソコンでは画面上の スタートメニュー を操作して電源を終了します。

まず、画面左下のスタートメニューにマウスの矢印を合わせ、クリックして開きます。次に、電源アイコンにマウスを合わせます。

スタートメニューを開く

「電源」という文字が表示されたらクリックします。すると、電源に関するメニューが現れます。

電源メニューを開く

メニューにはスリープ、シャットダウン、再起動の 3 つが並んでいるはずです。それぞれの意味を知っておきましょう。

シャットダウン

PC を終了するときの最も基本的な手順です。

これをクリックしてすぐに電源が切れるのではなく、少しだけ終了に関する処理が行われた後に電源が切れます。
完全に終了しないうちにノートパソコンを閉じたり、収納したりするのは避けましょう。

再起動

再起動とは、シャットダウンにしたあとにもう 1 回起動することです。
新しいアプリをインストールした後に内容を反映させる時や、パソコンの動作が不安定な時にリセットする目的などで使われます。

シャットダウンと違うのは、電源が切れたあとに自動的に再び電源がオンになることです。

スリープ

スリープとは、PC で作業中の内容（開いているアプリなど）を保持したまま、PC の動作を止める機能です。
シャットダウンに比べて、作業を素早く再開できるメリットがあります。
なお、スリープ中は完全に電源が切れているわけではなく、ほぼ電力を使わない状態で電源が入っています。

また、ノートパソコンは蓋（ふた）を閉じることでもスリープになります。

スリープの注意事項

スリープは作業途中の状態を保存したまま終了できる便利な機能ですが、時々動作が不安定になる場合があります。
そういった状況に遭遇したときは再起動をするとよいでしょう。

Windows のアップデート

Windows は定期的にアップデート（最新版への更新）が行われます。アップデートを行うことで Windows の不具合が修正されたり、安全性が向上したりします。

アップデートが利用できるときは電源メニューに以下のような「更新してシャットダウン」といった表示が現れます。

アップデートがある場合の表示

この状態で「更新してシャットダウン」をクリックすると、通常のシャットダウンや再起動の処理に加えて、アップデートの処理が自動で行われます。
ただし、通常のシャットダウンより時間が掛かるのでご注意ください。

パソコンの強制終了

パソコンの動作がおかしくなってしまった場合、強制終了しなければならないときがあります。手順を覚えておきましょう。いま試す必要はありません。
なお、強制終了をすると、作業中のデータが失われることがあります。

まず、キーボードの Ctrl キー と Alt キー と Delete キーを同時に押します（Delete キーは Del と略されていることもあります）。
その状態で画面右下に現れる電源アイコンをクリックし、シャットダウン をクリックします。

それでも動かない場合

この手順による強制終了はデータ破損など、トラブルの原因になることもあります。PC が操作を受け付けなくなるなど、やむを得ない場合のみに使用してください。

キーボードやマウスの操作をしても全く動かない場合、PC の電源ボタンを長押しします。
電源が切れるまで長押しし続けてください。
