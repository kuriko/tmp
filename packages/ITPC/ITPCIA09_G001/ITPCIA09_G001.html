パソコンに接続できる機器について学ぼう

様々な機器を接続し、必要に応じて機能を拡張できるのがパソコンの強みです。

どのような機器が接続できるのか、どうやって接続するのかについて見ていきましょう。
今回はたくさんの内容が出てきますが、すべて覚える必要はありません。「図鑑」のようにとらえて、気楽に眺めましょう。
まずは一通り読んでみて、実際に機器を接続するときに迷ったら再び読むとよいでしょう。

なお、機器はデバイスともいいます。

デバイスとパソコンを接続する方法（規格・端子）

デバイスを使うには、パソコンとデバイスを接続しなければいけません。
有線か無線か、有線ならどの手段を使うかなど、接続手段には様々な種類があります。

有線接続

USB

USB（ユーエスビー）はUniversal Serial Bus （ユニバーサル・シリアル・バス）の略です。
多くのデバイスが USB によってパソコンと接続できます。

USBにはいくつかの種類があるので、のちほど詳しく説明します。

Thunderbolt

Thunderbolt（サンダーボルト） は Mac など一部のパソコンに搭載されている端子です。

最近用いられているものは Thunderbolt 2 と Thunderbolt 3 があります。

Thunderbolt 2 は後述する Mini Display Port と、Thunderbolt 3 は USB Type-C と同じ端子を採用しています。そのため、Thunderbolt の端子を USB や DisplayPort の端子と同じように扱うこともできます。
ただし、逆に一般的な USB や Mini DisplayPort の端子を Thunderbolt として扱うことはできません。

音声端子

音声に関する機器を接続する端子です。パソコンに装備されているものは一般的に直径が 3.5 mm の端子で、「（オーディオ）ミニプラグ」などとも言われます。

ヘッドホン・イヤホン・スピーカーといった 再生機器 と、マイク（録音機器）で形状が同じです。挿し間違えないように気をつけましょう。
多くの場合はヘッドホンやマイクのアイコンが描かれており、どちらの端子か分かるようになっています。

HDMI

HDMI（エイチ・ディー・エム・アイ）は動画と音声を同時に転送する規格です。テレビやゲーム機などでも利用されています。
パソコンに装備されているのは HDMI の出力端子で、パソコンが出力する映像を別の画面やプロジェクターなどに出力するためのものです。ゲーム機などから出力される映像をパソコンに入力する場合は「キャプチャボード」と呼ばれるデバイスが必要です。

ちなみに、High-Definition Multimedia Interface（ハイ・デフィニション・マルチメディア・インターフェイス）の略で、「高精細度マルチメディアインターフェース」という意味ですが覚える必要はありません。

Display Port / Mini DisplayPort

Display Port（ディスプレイ・ポート）は映像を転送するための規格です。DPと略されることもあります。
HDMI と形状が似ているので要注意です。

なお、ノートパソコン向けに省スペースな Mini DisplayPort（Mini DP）と呼ばれる端子もあります。

一般的なテレビやゲーム機などではあまり採用されておらず、どちらかというと業務用寄りの端子です。
たとえば、一部の VR 機器は DisplayPort を採用しているものもあります。

D-Sub（VGA）

D-Sub（ディー・サブ） あるいは VGA（ブイ・ジー・エー）は映像を転送するための端子です。アナログRGB端子と呼ばれることもあります。
昔はよく使われていた古い規格なので高画質の映像を転送することはできませんが、古いプロジェクター等ではまだ使われることがあります。

LAN端子

LAN（ラン）端子 は Wi-Fi ではなく有線でネットワークに接続するときに使う端子です。
多くの場合は有線よりも Wi-Fi のほうが便利ですが、より高速で安定した通信が必要な場合に有線 LAN は利用されます。

LAN ケーブルには「カテゴリ」とよばれる分類があり、カテゴリによって転送速度が違います。

LAN ケーブルのカテゴリ

家電量販店で LAN ケーブルの売り場に行くと、「CAT 7」のような表記を見かけます。CAT は カテゴリ の略です。
一般的に、カテゴリの数字が大きいほど高速な通信に対応しています。

  • カテゴリ5e : 1Gbps
  • カテゴリ6 : 1Gbps（5eより安定）
  • カテゴリ6A : 10Gbps
  • カテゴリ7 : 10Gbps（6Aより安定）
  • カテゴリ8 : 40Gbps

ただし、LAN ケーブルの速度だけでなく、パソコンやネットワークの機械がすべて対応していないとその通信速度を出せません。
つまり、パソコンが 1Gbps にしか対応していないなら、カテゴリ 8 のケーブルを買ってもあまり意味がありません。

普通に使うのであれば カテゴリ6 や カテゴリ6A を買っておけば十分と言えます。
なぜなら、2021年現在のパソコンやネットワークの主流は 1Gbps であり、10Gbps 以上はまだ業務用のものしか存在しないからです。

無線接続

Bluetooth

Bluetooth（ブルートゥース）は、スマホでも活用されている無線接続の規格です。
近年のノートパソコンは Bluetooth に対応している機種も多く、対応していれば イヤホン・ヘッドフォン・キーボード・マウスなどを無線で接続することができます。

Bluetooth は英語で「青い歯」という意味ですが、これは 10 世紀にデンマークとノルウェーを話し合いによって平和的に統一したハーラル1世（通称「青歯王」）にちなんでいます。
「複数の規格を作ってメーカー同士で争うのではなく、Bluetooth で平和的に統一しよう」といった願いが込められた名前だと言われています。

コラム: Bluetooth のロゴ

米印と「B」が組み合わさったような Bluetooth のロゴマークですが、これは当時使われていた「ルーン文字」で表したハーラル1世のイニシャル（ᚼᛒ＝H.B）を組み合わせたものです。

USB の種類について

USB は 1990 年代から利用されている端子であり近年も改良が続いているため、特に多くのバリエーションがあります。
バリエーションは バージョンと端子の形の 2 つの観点から分類できます。

特に近年は規格がどんどん複雑になっているので、すべての規格を正確に理解する必要はありません。大まかに以下のような認識をしていれば大丈夫です。

バージョンで見る USB

USB は大きく分けて 4 世代あり、通信速度が異なります。
互換性があるため、新しい世代のものを古い世代のものに接続して使うことができます。ただし、速度は古いものになります。
たとえば、 USB 2.0 の端子しかない古いパソコンに USB 3.0 に対応したケーブルやデバイスを接続した場合、USB 2.0 の速度で通信が行われます。

USB 1.0 / USB 1.1

1990 年代に発表された最も古いバージョンです。12 Mbps という速度で通信できます。

USB 2.0

2000 年に発表されたバージョンで、USB 1.1 よりも高速な通信（480 Mbps）ができるようになりました。

USB 3.0

USB 3.0 は 2008 年に発表されました。USB 2.0 よりも更に高速な通信（最大 5 Gbps）ができるようになりました。
区別のために端子やコネクタが青くなっています。また、端子やケーブルに SS（Super Speed、スーパースピード）と記載されていることもあります。

その後も USB 3.1 や USB 3.2 といった拡張がなされ、高速化がどんどん進んでいます。

USB4

2019 年に発表された規格です。さらに高速で通信できるとされています。
規格が発表されたばかりなので、USB4 に対応したデバイスが登場するのはもう少し先になるでしょう。

端子の形状で見る USB

バージョンだけでなく、端子の形状も種類があります。
ここでは主要な端子について見てみましょう。

USB Type-A（タイプA）

USB の中では最も基本的な端子です。裏表が分かりづらい事に要注意です。
USB-A と書かれることもあります。

USB Type-B（タイプB）

USBの中では比較的大型の端子です。プリンターなど大型の機器で使われます。
USB-B と書かれることもあります。

USB Type-C（タイプC）

スマホなどでも採用されている端子で、裏表どちらでも差し込めるのがメリットです。
USB-C と書かれることもあります。

micro USB（マイクロUSB）

スマホや小型のデバイスなどでよく採用されています。
近年は USB Type-C に切り替わりつつあり、今後は数を減らしていくと思われます。

参考: mini USB（ミニ USB）

最近は見かけなくなりましたが、これも小型の端子です。micro USB より少し大きめのサイズです。
古いデバイスなどで使われていることがあります。

実際の USB ケーブルを見てみよう

上記の分類を学んだら、実際に USB ケーブルや端子を見てみましょう。
もし手元に実際の USB ケーブルがあるならそれを見ても良いですし、Amazon などのネット通販サイトで「USB ケーブル」と検索して説明を読んでみてもよいでしょう。

片方の端子が USB Type-C で、もう片方の端子が USB Type-A の USB 3.0 に対応したケーブルだということが分かります。

USB の変換

//TODO 説明イラスト: USB Type-C しかないPCに、変換アダプタを使って USB Type-A のデバイス（USBメモリ）を差し込む様子（PC全体が描かれる必要はなく、端子部分だけをクローズアップ）

端子の形は違っても同じ信号をやり取りしているため、USB は変換することができます。変換するための装置を「変換アダプタ」といいます。
たとえば、Type-A と Type-C の変換コネクタを買えば、Type-C の端子しかないパソコンに Type-A のケーブルを差し込むことができます。

ただし、差し込むことができるからといって必ず動作するわけではない事に注意が必要です。
たとえば、スマホの USB Type-C 端子に USB Type-A 変換アダプタを使って、パソコン専用のデバイスを接続しても動作するとは限りません。

ドライバ

ドライバとは、デバイスをパソコンに認識させるための特殊なアプリのようなものです。
近年はデバイスが接続された際に Windows や macOS などの OS がドライバを自動的に探してくれる事が多くなっています。

しかし、特殊なデバイスなどを使用するときは OS が探しきれず、ドライバを手動でインストールする必要が出てくることもあります。そのような場合はデバイスの説明書や Web サイトなどを参考に、自分でインストールしてみましょう。
