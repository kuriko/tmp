Zoom のアカウント作成

Google Chrome で Zoom の公式サイト（https://zoom.us/）にアクセスします。
画面右上の サインアップは無料です をクリックしましょう。「サインアップ」は登録という意味です。

年齢確認のため、あなたの誕生日を入力します。

Zoom に使うメールアドレスを入力し、サインアップをクリックします。
特にこだわりがなければ Gmail のメールアドレスなどでよいでしょう。

不正登録を防ぐため、画像認証が行われます。指示に従って進めてください。

以下の画面が表示されたら、メールを確認してください。
たとえば Gmail のアドレスを登録した人は、新しいタブで Gmail を開いてください。

以下のようなメールが届いているはずです。アカウントをアクティベートをクリックします。

Zoom の登録をする画面が再び現れます。
名（下の名前）、姓（苗字）、パスワード、パスワードの確認 の4つの入力欄があるので、全て入力してください。
「初頭または中等教育機関……」は いいえ を選んでください。
入力し終わったら 続ける をクリックします。

仲間を増やしましょうの画面では、手順をスキップするをクリックします。

テストミーティングを開始の画面では マイアカウントへ をクリックします。

Zoom のインストールと実行

マイアカウント の画面で、画面の右上にある ミーティングを開催する をクリックします。
ビデオはオフ をクリックします。

Zoom のインストーラーがダウンロードされます。
画面の左下に zoom.pkg というファイルが表示されるので、それをクリックします。

なお、この際 Google Chrome は閉じないでそのまま開いておきます。

インストーラーが起動します。Zoom の紹介文が表示されるので、続ける をクリックします。

インストールをクリックします。

インストールが自動で実行されるので、完了を待ちます。

完了したら 閉じる をクリックします。

インストーラーは今後使わないので、ゴミ箱に入れる をクリックします。

OK をクリックします。

開きっぱなしにしていた Google Chrome の画面で ミーティングを起動 をクリックします。

zoom.us を開きますか？ という画面が出てくるので、zoom.us を開く をクリックします。
なお、以下の画面では us05web と記載されていますが、この部分の表示は人によって異なります。

すると、先ほどインストールした Zoom のアプリが起動します。
マイクの権限が必要なので、OK をクリックします。マイクは後からオフにできるのでご安心ください。

コンピューターオーディオに参加する をクリックします。

カメラの権限が必要なので、OK をクリックします。カメラは後からオフにできるのでご安心ください。

バーチャル背景を設定しよう

バーチャル背景とは、人物のみを認識して背景部分を画像や動画に差し替えてくれる機能です。
カメラをオンにして顔は見せたいけれど、背景（部屋）を見せたくないという場合に活用できる機能です。

バーチャル背景を利用するには、まず画像を用意する必要があります。また、動画をリアルタイムで加工するため、パソコンの性能によっては動作しないこともあります。

音声トラブルが起きた場合に備えよう

喋っているのに相手に聞こえない、または相手の声が聞こえないというときは、誤ったマイク・スピーカーが選択されている可能性があります。

Zoom 会議への参加

他の人が作成した会議に参加する手順を学びましょう。

URL を確認する

まずは Zoom の URL を確認しましょう。Zoom では会議 1 つに対して 1 つの URL が用意されます。
以下のような案内文がメールやチャットなどで送られてきたり、オンライン授業のシステムに掲載されていたりします。

○○さんがあなたを予約されたZoomミーティングに招待しています。

トピック: 定例ミーティング
時間: 毎日夕方に行います

Zoomミーティングに参加する
https://zoom.us/j/xxxxxxxxx

ミーティングID: xxx xxx xxx
パスワード: xxxxxxxxx

https://zoom.us/ から始まる URL をクリックしましょう。
ミーティングID はこの URL に含まれているため、使う必要はありません。また、パスワード は入力が必要な場合とそうでない場合があるので、求められた場合は案内文に掲載されているものを使いましょう。

パスワードの入力・待機室

パスワードが設定されている場合、パスワードを入力しないと参加できません。

また、「待機室」とよばれる仕組みが設定されている場合もあります。待機室が設定されている場合は、会議に参加する場合に主催者が 1 人ずつ許可する仕組みになっています。

複数人が参加する会議での操作

基本的には先ほど一人で練習したときと同じです。
ただし、複数人のカメラや名前をどのように表示するか切り替えできます。

ギャラリービュー

自分を含む複数人を同じサイズで並べて表示する状態を「ギャラリービュー」といいます。

ギャラリービュー

ギャラリービューの状態でウィンドウ右上にある スピーカービュー ボタンをクリックすると、スピーカービューに切り替えられます。

スピーカービューへの切り替えボタン

スピーカービュー

いま喋っている人を拡大して表示する状態を「スピーカービュー」といいます。

スピーカービュー

スピーカービューの状態でウィンドウ右上にある ギャラリービュー ボタンをクリックすると、ギャラリービューに切り替えられます。

Zoom 会議を作成する

自分が会議を主催する場合の手順です。

Google Chrome で Zoomのサイト（https://zoom.us/）を開き、右上の マイアカウント をクリックします。
「ミーティングを主催する」ではないので気をつけてください。

続いて、ミーティングをスケジュールする をクリックします。

以下のような、ミーティング作成ページが表示されます。1つずつ入力していきます。

まず、トピック を決めます。トピックとは「話題」「議題」を意味する英単語で、ミーティングのタイトルのことです。
ここでは「プロジェクト進行状況の打ち合わせ」としてみます。
説明は何も入力しなくて大丈夫です。余談ですが、ここで書かれている「ウェビナー」とは、ウェブ + セミナー の造語で、ネットで行われるセミナーのことを指します。

ミーティングの 開催日時 と 所要時間 を決めます。
タイムゾーンとは時差のことで、(GMT+9:00) 大阪、札幌、東京 を選ぶと日本時間になります。

なお、Zoom の無料プランの場合、3人以上が参加するオンラインミーティングは連続40分までしか実施できませんのでご注意ください。

会社や学校のアカウントの場合

会社や学校で有料プランが使用されている場合、40 分の時間制限はありません。


ミーティング ID は 自動的に生成 を選びます。

セキュリティに関する設定です。
パスコード はパスワードと同じ意味で、入室の際に必要なパスワードをここで決めます。初期状態でランダムな文字列が決められているので、そのままにしておいても良いでしょう。
また、待機室をオンにすると、会議の管理者（ここではあなた）が会議に参加しようとしている人の名前を見てから入室許可を出すことができます。これにより、見知らぬ人が会議に入ってくることを防止できます。

ビデオ設定では、初期状態でカメラをオンにするかどうか選べます。
最初からオンになっていると準備ができず困る人も多いので、オフにしておくと良いでしょう。

続いて、オプションの設定です。

ホストの前の参加を有効にする のチェックを入れておくと、会議主催者（あなた）が Zoom を起動する前に、他の参加者がミーティングに参加して待機することができます。
自分がいなくても会議を始めておいてほしいという場合にはこれをオンにするとよいでしょう。なお、ホストとは会議主催者を意味します。

入室時に参加者をミュートにする のチェックを入れておくと、会議に参加するのマイクが最初の時点ではオフになります。
オンライン授業や発表などで、先生や発表者以外のマイクがオンになっていると困る場合に使いましょう。

ミーティングをローカルコンピュータに自動的にレコーディング のチェックを入れておくと、ミーティングの様子が自動的に動画として記録されます。

すべて入力したら、次に進みます。

以下のような画面が表示されたら作成完了です。このミーティングを開始をクリックするとミーティングを開始できます。

また、招待リンク の欄にある 招待状のコピー をクリックすると、ミーティングへの招待リンクを手軽にコピーできる画面が表示されます。
以下の画面で下の方の ミーティングの招待状をコピー のボタンをクリックすると、自動で文章がコピーされます。

招待状をコピーしたら、メールやチャットなどに貼り付けして、ミーティングに参加してほしい人に情報を共有しましょう。
なお、この招待状リンクを使うとパスワードなしでミーティングに参加できます。URL が外部の人に流出しないように気をつけてください。

その他のビデオ会議ツール

Zoom 以外にもビデオ会議のツールは存在しています。会社や学校によってはそちらを使うこともあるので、名前だけでも知っておくとよいでしょう。

  • Cisco Webex（シスコ・ウェベックス）
  • Google Meet（グーグル・ミート）
  • Microsoft Teams（マイクロソフト・チームズ）
  • Skype（スカイプ）
