Windows

Windows の画面（例）

Windows（ウィンドウズ）はマイクロソフト社によって開発・販売されている OS です。
現在の最新版は Windows 10（ウィンドウズ・テン）です。

Windows のバージョン

近年のバージョンを見てみましょう。
ちなみに「Windows 9」は存在しません。

バージョン名 発売時期 サポート終了時期
Windows 7
（セブン）		 2009年9月 2020年1月
Windows 8
（エイト）		 2012年8月 2016年1月
Windows 8.1 2013年8月 2023年1月
Windows 10
（テン）		 2015年7月 -

Windows の各バージョンにはサポート期間があります。サポート期間中はセキュリティのためのアップデートが行われますが、この期間が終了するとアップデートが行われなくなります。サポートが終了した OS はコンピューターウイルスに感染したり、個人情報が盗まれたりといったリスクが高いので、使用してはいけません。

Windows 8 シリーズまでは数年に 1 回のペースで新しい Windows が発売されていました。そのため、従来の Windows ユーザーは数年に 1 回のペースで Windows を買い換えないといけませんでした。
しかし、現在のマイクロソフトは次の「Windows 11」などを発売することはなく、Windows 10を常にアップデートし続けていくという方針をとっています。そのため、Windows 10が搭載されたパソコンを購入すれば 新しく OS を購入する必要はなく、安心して使い続けられるとされています。

この講座では、Windows 10 を対象に解説していきます。

macOS

macOS の画面（例）

macOS（マックオーエス）はアップル社のパソコン「Mac」に搭載されている OS です。以前は OS X（オーエス・テン）、Mac OS X（マックオーエス・テン）という名称でしたが、2016 年に名前が変わりました。

現在の最新版は macOS 11 Big Sur（マックオーエス・イレブン・ビッグサー）です。

macOS のバージョン

近年リリースされた macOS のバージョンを見てみましょう。
macOS は各バージョンに英語の名称がついています。これらの名称はアメリカの地名に由来します。

およそ 1 年おきに新しいバージョンの macOS がリリースされています。新しいバージョンへは無料で更新（アップグレード）できます。

Windows と異なり、macOS はサポート期間が明確に示されていません。しかし、最新版から 2 ～ 3 つ前のバージョンまでしか最新のアップデートが提供されていないので、なるべく最新版を使うことが望ましいでしょう。

コラム: 激しく変化する macOS

macOS はバージョンが更新されても、見た目があまり変わりません。一見すると大きな新機能はないのではないかと思ってしまいますが、中身はガラッと変わっていることが多いのです。
そのため、特殊なソフトなどは新バージョンで突然動作しなくなることもあります。
一方で Windows は新バージョンで見た目が大きく変わっても、内部的な変化は macOS ほど大きくありません。
Windows は企業で利用されていることが多く、中身を変えてしまうと業務で使用されていた専用アプリなどの動作に影響を及ぼしてしまいます。そのため、マイクロソフト社は「過去のアプリも最新 OS で使えること」を目標にしています。
過去のアプリが使えなくなってしまったとしても「理想的な最新 OS」を追求するアップル社と、過去のアプリを考慮して大きな変化を避けるマイクロソフト社。それぞれの方針の違いも興味深いですね。

まとめ

  • Windows と macOS にはそれぞれバージョンがある。
  • 古いバージョンはサポートされなくなり、安全ではなくなるので、なるべく最新バージョンを使うべきである。
