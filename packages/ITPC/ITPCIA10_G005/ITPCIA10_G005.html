ファイルをバックアップしよう（Mac）

パソコンはいつ壊れてしまうか分かりません。また、誤操作やコンピューターウイルスなどにより、ファイルが消えてしまう可能性もあります。
そういった時にデータを守ってくれるのが バックアップ です。

  • バックアップとは？
  • バックアップの方法
  • クラウドを使ったバックアップ

バックアップとは？

バックアップ（backup）とは、英語で「支援」や「代わりのもの」を意味する言葉です。 パソコンのファイルについてこの言葉を使うときは、「パソコンが壊れてしまったときに備えて、データを別のところにもコピーしておく」という意味になります。

何をバックアップするか

パソコンの中には大量のファイルがありますが、その全てをバックアップするのは大変です。

自分で作成・撮影したファイルは必ずバックアップする

自分で作成した文書やプレゼン、自分で撮影した写真や動画などは必ずバックアップしましょう。
なぜなら、これらはパソコンが壊れた時に同じものを用意できないからです。特に、思い出を記録した写真や動画などは同じものを作り上げることはできません。

ネットからダウンロードしたファイルは場合による

たとえば「ネットで見つけたちょっと面白い画像」などのように、重要度が低いファイルはバックアップする必要はないでしょう。

逆に、ネットから入手したものでも、勉強や仕事で使う重要なファイルはバックアップしておきましょう。
なぜなら、データを配布しているサイトなどが消滅するなどして、再度ダウンロードできなくなる場合があるからです。

アプリはバックアップしない

ほとんどの場合、アプリは公式サイトや App Store などから容易に再ダウンロードできます。
そのため、アプリはバックアップせず、必要になったら再ダウンロードするようにしましょう。

バックアップに関するよくある誤解

誤解 1: 「単にコピーすればよい」

同じパソコン内でファイルをコピーをしても意味がありません。なぜなら、パソコンが壊れると、そこに入っているファイルは全て取り出せなくなってしまうためです。
全く別の場所（外付けのHDDやクラウドなど）に保存する必要があります。

誤解 2:「1 回だけバックアップすれば良い」

バックアップはその時点でのコピーに過ぎません。1 回コピーしただけでは、復元できるデータも古いものになってしまいます。
定期的に最新のデータをバックアップする必要があります。

誤解 3:「USB メモリに保存する」

USB メモリはその特性上、データが消えてしまいやすく、長期保存には向いていません。
どこにもバックアップしないよりはマシですが、重要なデータはクラウドや外付け HDD などに保存しましょう。

バックアップの方法

個人のパソコンのデータをバックアップする場合、主な手段は 2 つあります。

  • クラウド（Google ドライブ）
  • 外付けHDD

それぞれの手段を学んでいきましょう。

クラウドを使ったバックアップ

一番手軽なのは、クラウドサービスを使う方法です。この講座では Google ドライブを使う方法を学びます。
書類やレポート、卒論などのデータなどをバックアップする場合におすすめです。

メリット

  • 基本は無料で利用できる。
  • Google のサーバーに保存されるため、故障リスクがほとんどない。
  • 外出先などからでもデータにアクセスできる。

デメリット

  • 15 GB を超える容量の場合、毎月の課金が必要になる。
  • 会社や学校の Google Workspace を使う場合、在籍中しか利用できない。転職や卒業をするとデータが消えてしまう。
  • ネット回線が遅い場合、データの転送に時間がかかる。
  • Google の方針が変わる可能性もあり、無料で使える容量や料金などが変わるおそれがある。

バックアップ・復元方法

9 章の「Google ドライブを使ってデータを保存・共有しよう」で学んだ手順を使い、バックアップするべきファイルをアップロードします。

復元する場合は、Google ドライブからダウンロードします。

外付けHDDへのバックアップ

続いて、外付け HDD（ハードディスクドライブ）を購入してそれにバックアップする方法をご紹介します。
動画や大量の写真など、容量の大きいファイルを保管する場合におすすめです。

メリット

  • 一度 HDD を買ってしまえば、大容量の保存領域を好きに利用できる。
  • USB接続のため、ネット回線の速度と関係なく高速に利用できる。

デメリット

  • 購入時に数千円の費用がかかる。
  • HDD を自分で管理する必要があるため、壊さないように気をつける必要がある。

バックアップ方法

Mac で使うための初期化

2021 年現在、Windows の方が普及しているため、多くの外付け HDD は Windows のファイルシステムに合わせて作られています。そのため、初期状態では Mac で使えないものがあります。
ファイルシステムとは、Windows や Mac といった OS がストレージにファイルを保存する仕組みです。

Windows 用のものでも、初期化（フォーマット）をすれば簡単に Mac 用に変換できます。
なお、この作業は新しい HDD を Mac で使うとき、最初に 1 回だけ実行するものです。次回以降は接続するだけで、普通に使えるようになります。

まずは HDD を Mac に USB で接続します。

続いて、Finderを開きます。

画面一番上のツールバーから 移動 をクリックし、 ユーティリティ をクリックします。

ディスクユーティリティ をクリックします。

左側で 外部 に表示されているものをクリックします（「内蔵」と間違えないようにしてください）。製品によってここに表示される名前は異なりますが、今回の例では「HDPH-UT」となっています。
すると、右側にその HDD の情報が表示されます。

ここに Windows NT ファイルシステム と表示されている場合、それは Windows 用であるという意味です。Mac 用に変換する必要があります。
もしここに exFAT や APFS などと表示されていた場合、既に Mac に対応したものとなっているので、初期化せずに次のステップに進んでください。

では、Windows NT ファイルシステム になっている場合は以下の手順で初期化していきましょう。
画面の上の方にある 消去 をクリックします。

注意：初期化をすると、外付け HDD にあるデータは消えてしまいます。新品の HDD なら気にする必要はありませんが、別のパソコンで使っていた HDD を使うときは、データを別のところにコピーしておきましょう。

以下のような画面が表示されます。名前はそのままで、フォーマット の Windows NT ファイルシステム（NTFS） となっている部分の右側にあるボタンをクリックしましょう。

exFAT をクリックします。

exFAT（イーエックス・ファット） は Windows でも Mac でも読み書きできるファイルシステムです。
Mac 専用の「APFS」よりは読み書きの速度は低いのですが、バックアップは他のパソコン（Windows）で読み出すこともあるため、exFAT にしておくことをおすすめします。

exFAT になっていることを確認したら、消去 をクリックします。

少し時間がかかるので、そのまま待ちます。この際、絶対に HDD を Mac から取り外してはいけません。

以下のような画面になったら 完了 をクリックします。

先程の画面でも exFAT になっていることが確認できたら、ディスクユーティリティを終了させましょう。

バックアップする

初期化が終わった後や、2 回目以降に HDD を繋ぐ場合の手順です。

Finder を開きます。

バックアップしたいフォルダを開きます。

バックアップ対象のフォルダやファイルをコピーします。

Finder の左側にある 場所 から、外付け HDD の名前をクリックします。
ここに表示される名前は HDD の製品によって異なりますが、以下のスクリーンショットの場合は「HDPH-UT」となっている部分が外付け HDD です。

外付け HDD が開いたら、何もない部分を 右クリック して ペースト をします。

無事にファイルやフォルダがコピーできたらバックアップ成功です。

2 回目以降のバックアップ

最新のデータをきちんと保護するには。定期的にファイル・フォルダをバックアップする必要があります。
その際もここまで同じ手順で構いません（ただし、2回目以降の場合はフォーマットは不要です）。

ただし、既に外付け HDD に保存されているものと同じ名前のファイル・フォルダをコピーすると、以下のようなメッセージが表示されます。
上書きして、古いものを保存しておく必要がない場合は 置き換える をクリックしてください。

もし、古いものも保存しておく必要がある場合は 中止 をクリックして、古いほうのファイル・フォルダ名を別の名前に変更してください。
（同じ場所に同じ名前のファイル・フォルダを保存することはできないからです。）

取り外し

何もせずに HDD を取り外すと故障の原因となります。
Finder で取り出しボタンをクリックしてから取り外しましょう。

復元方法

新しいパソコンや代わりのパソコンに外付け HDD を USB で接続します。

操作方法は USB メモリなどと同様です。
