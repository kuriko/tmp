パソコンを構成する部品

パソコンは様々な部品を組み合わせて作られています。
それぞれの部品の特徴を見ていきましょう。

CPU

//TODO 説明イラスト: CPUは頭脳のようなもの

CPU（シーピーユー）とは、PC の頭脳に当たる部品です。 Central Processing Unit（セントラル・プロセッシング・ユニット、中央演算処理装置）の略称です。プロセッサ と呼ばれることもあります。

CPU (画像提供 : Intel)
動作周波数（クロック周波数）
ここはやや重要な内容です。もし余裕があれば理解しましょう。

//TODO 説明イラスト: 周波数が高いほど動作が速い

かんたんに言ってしまうと、CPU が仕事をする速度です。単位は Hz（ヘルツ）です。
周波数が低いほど遅く、高いほど速く動作します。 たとえば、周波数が 1.1GHz（ギガヘルツ） より、2.6 GHz（ギガヘルツ） の方が高性能だと言えます。

周波数が高いと、より多くの電力を消費したり、熱を発したりします。
そのため、バッテリで動作する事が多く、省エネを求められる ノートパソコン用の CPU は周波数が低めに抑えられています。

コア数
ここはやや重要な内容です。もし余裕があれば理解しましょう。

//TODO 説明イラスト: マルチコアCPUはより多くの仕事を同時に処理できる

上記の写真は一見すると 1 つのCPUに見えますが、中には複数の CPU が詰まっています。
この CPU の数が「コア数」です。

昔は 1 コア が基本で、クロック周波数を上げることによって高性能化が図られてきました。しかし、周波数を上げるだけでは技術的に限界があるため、CPU を増やすという手法がとられるようになりました。
コアが多くなればなるほど、同時に処理できる仕事が増えます。

2 つ以上のコアがある CPU のことを「マルチコア CPU」といいます。
以下のような名称も知っておくとよいでしょう。

  • 1 つのコアしかない CPU: シングルコア CPU
  • 2 つのコアがある CPU: デュアルコア CPU
  • 4 つのコアがある CPU: クアッドコア CPU
  • 6 つのコアがある CPU: ヘキサコア CPU
  • 8 つのコアがある CPU: オクタコア CPU

メモリ

ここは非常に重要な内容です。きちんと理解し、用語も覚えましょう！

//TODO 説明イラスト: メモリは机のようなもの

メモリは、処理をするためのアプリやデータを一時的に記憶するためのパーツです。
CPUを頭脳とすると、机のようなものです。

机が広ければ広いほど、一度に多くのデータを広げることができるのでスムーズに処理ができます。机が狭い場合、多くのデータを載せることができなくなり、処理速度が低下します。

ストレージ

ここは非常に重要な内容です。きちんと理解し、用語も覚えましょう！

//TODO 説明イラスト: ストレージは倉庫のようなもの（メモリとは違う）

ストレージ とは、プログラムやデータを記録する装置です。
メモリと違うのは、PC の電源を切ってもストレージの内容は消えない点です。例えるならば本棚や倉庫のようなものです。

ストレージは主に 2 種類に分けられます。ノートパソコンの多くはどちらか 1 種類を搭載しています。

HDD（ハードディスクドライブ）

//TODO 説明イラスト: HDDのメリットとデメリット（下記の通り）

磁気を用いてデータを記録します。内部には磁気ディスク（円盤）が重ね合わせた状態で入っています。

  • メリット: 安価で大容量。
  • デメリット: 読み書きする速度が遅い。落下などの衝撃が加わるとデータが消失する恐れがある。
SSD（ソリッドステートドライブ）

//TODO 説明イラスト: SSDのメリットとデメリット（下記の通り）

電子回路にデータを記録します。

  • メリット: 読み書きが高速。衝撃に強い。
  • デメリット: 大容量にすると高価。
SSD (ソリッドステートドライブ) (画像提供 : Intel)

GPU

ここはやや重要な内容です。もし余裕があれば理解しましょう。

//TODO 説明イラスト: GPU は画像処理に特化したデバイス

GPU （ジーピーユー） とは、画像（特に3D）に特化した処理を行う部品で、Graphics Processing Unit（グラフィックス・プロセッシング・ユニット）の略称です。

元々はゲームなどの CG（コンピューター・グラフィックス）を高速で処理するために開発されています。

基本的には PC でゲームをプレイする人や、3Dコンピューター・グラフィックス、映像を制作する人が重視するべき部品です。
ネットを閲覧したり、PC を学習に使う程度であればあまり GPU に関するスペックを気にする必要性はありません。

まとめ

  • パソコンは様々な部品を組み合わせて作られている。
  • CPU
    • パソコンの頭脳。
    • クロック周波数が大きいほど処理が速い。
    • コア数が多いほど同時に複数の処理ができる。
  • メモリ
    • 机にあたるもの。
    • 容量が大きいほど作業がスムーズ。
    • データを保存する場所ではない。
  • ストレージ
    • 本棚や倉庫にあたるもの。
    • データを保存しておく場所。
    • SSD（高速、高価）とHDD（安価、大容量）がある。
  • GPU
    • 画像処理専門の部品。
    • 3D ゲームをする場合などに重視する必要がある。
    • 通常の用途ならそこまで気にしなくても大丈夫。
