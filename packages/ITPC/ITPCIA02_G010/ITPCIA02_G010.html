圧縮ファイルを扱ってみよう

容量の大きいファイルは 圧縮（あっしゅく）できます。
Web サイトに掲載したり、人にメールなどで送ったりする場合に便利です。

エクスプローラーは ZIP（ジップ）という形式での圧縮に対応しています。

  • 圧縮
  • ZIP ファイルを展開しよう

圧縮

ファイルを圧縮しよう

ドキュメント を開き、勉強 フォルダを開きましょう。続いて、さらにその中にある パソコン フォルダを開きましょう。

前回の「ファイルやフォルダ操作の基本を学ぼう」で作成した、「ファイルの練習」というファイルを 右クリック し、送るにマウスカーソルを合わせます。そのまま 圧縮(zip 形式) フォルダー をクリックします。

「送る」から「圧縮 (zip 形式) フォルダー」を選択

すると、「ファイルの練習」という名前でチャックの付いたフォルダアイコンが登場したはずです。これが ZIPファイル（ZIP形式フォルダ）です。

「ファイルの練習」という名前の ZIP ファイルが作成された

作成された ZIP ファイルをダブルクリックしましょう。
すると、普通のフォルダのように中身を表示できます。ただし、エクスプローラーの上部には 圧縮フォルダツール というものが表示されているはずです。

「圧縮フォルダツール」と表示されている

フォルダを圧縮しよう

では、ドキュメント に戻りましょう。
今度はフォルダの圧縮を試してみましょう。

「送る」から「圧縮 (zip 形式) フォルダー」を選択

ドキュメントで、「勉強」というフォルダを 右クリック し、送るにマウスカーソルを合わせます。そのまま 圧縮(zip 形式) フォルダー をクリックします。

フォルダが圧縮された

ファイルの時と同様に、フォルダが圧縮されました。

ZIP ファイルを展開しよう

続いて、ZIP ファイルを 展開（てんかい） してみましょう。展開とは、圧縮されたものを普通のファイル・フォルダに戻すことです。冷凍品に例えて解凍（かいとう）と呼ばれることもあります。

ネットから入手したファイルや、他の人からメールで送られてきたファイルは ZIP 形式になっていることが多いため、展開操作を覚えておく必要があります。

先ほど圧縮結果を確認したときのように、ZIP ファイルの中身は通常のフォルダのように表示できます。（ただし、圧縮フォルダーツール が表示されます。）

「圧縮フォルダツール」と表示されている

しかし、圧縮された状態で中身のファイルを開こうとすると、トラブルの原因になります。
そのため、エクスプローラーで ZIP ファイルの中身を見るのは「どういうファイルが入っているんだろう？」といった内容の確認だけにして、きちんと 展開 してから利用するようにしましょう。

まず、ZIP ファイルを 右クリック し、すべて展開 をクリックします。

「すべて展開」を選択

圧縮 (ZIP 形式) フォルダーの展開 というウィンドウが現れるので、展開 をクリックします。
このとき、「完了時に展開されたファイルを表示する」にチェックを入れておくと、展開が終わったときにファイル（フォルダ）を表示できます。

「圧縮 (ZIP 形式) フォルダーの展開」ウィンドウ

また、ZIP ファイルをダブルクリックして開いた状態でも展開できます。
ZIP ファイルをエクスプローラーで開いているときは「圧縮フォルダツール」が上部に表示されます。

圧縮フォルダツールをクリックし、その中にある すべて展開 をクリックします。
これで右クリックした時と同じメニューが表示されます。

「圧縮フォルダツール」から「すべて展開」を選択

今回は自分で圧縮したフォルダを展開しましたが、実際には人から送られてきたファイルやダウンロードしたファイルに対してもこの展開操作をしていくことになります。

圧縮の注意事項

ファイルを圧縮には限度があるため、ファイルの種類によっては圧縮してもファイルサイズが小さくならないものもあります。
また、他の OS (macOSなど) で圧縮されたファイルを開くと、その OS 独自のファイルが混入したり、ファイル名が文字化けすることがあります。

コラム: ファイルサイズを小さくするだけじゃない

ZIP 圧縮の主要な目的はファイルサイズを小さくすることですが、複数のファイルを 1 つのファイルにまとめられるというメリットもあります。


また、電子メールにおいては、ファイルをメールに添付してやり取りできますが、フォルダの直接添付はできません。もしフォルダをまるごと添付したい場合、そのフォルダを圧縮して ZIP として添付する必要があります。
