電源のオン・オフと Mac のスペック確認

Mac での電源の操作を学びましょう。また、起動したあとに自分が使っている Mac のスペック（性能・バージョン）を確認してみましょう。

その前に、電源のアイコンを覚えておきましょう。
これは電子機器における 電源 のアイコンです。スマートフォンやゲーム機などでも目にした事があるのではないでしょうか。
一見するとリンゴのようにも見えますが、デジタルの 0（オフ） と 1（オン）を意味しているといわれています。

起動方法（スマホとの違い）

MacBook シリーズ（MacBook, MacBook Air, MacBook Pro）では、Mac の蓋（ふた）を開くと電源がオンになります。
もし、既に蓋（ふた）を開いている状態で電源がオフになっており、改めて電源を入れたい場合は電源ボタンを操作してください。

なお、電源ボタンを操作する場合は長押しする必要はありません。1 回カチッと押せば大丈夫です。

電源ボタンの場所

MacBook シリーズの一部では、キーボード右上にある指紋センサーが電源ボタンになっています。

ここで確認してほしいものが 2 つあります。

バージョン

確認してほしいものの 1 つは バージョン です。
バージョンといっても 11.0 のような数字ではなく、macOS に付けられた Big Sur（ビッグサー） や Catalina（カタリナ）といった名前が分かっていれば大丈夫です。

macOS のバージョン
macOS 10.12 Sierra（シエラ）
macOS 10.13 High Sierra（ハイ・シエラ）
macOS 10.14 Mojave（モハベ）
macOS 10.15 Catalina（カタリナ）
macOS 11 Big Sur（ビッグサー）

Catalina 以前と Big Sur では画面の見た目など異なっています。
そのため、この教材では Catalina 以前 と Big Sur で解説を分けている場所があります。自分が使っているバージョンに合ったものを読んでください。

プロセッサ

もう 1 つは プロセッサ です。これは CPU のことです。
詳細な型番を確認する必要はないのですが、ここに Intel と書かれているか、Apple と書かれているかを確認しておいてください。

Intel と書かれているものは Intel 製 CPU を搭載した 従来型の Mac 、Apple と書かれているものは Apple 製 CPU（Apple Silicon、アップルシリコン）を搭載した 新型の Mac です。
どちらも操作は変わりませんし、できることも同じです。ただし、Apple Silicon 搭載モデルに対応していないアプリもあるので、自分がどちらの Mac を使っているかを知っておくことが重要です。

スリープにする

スリープとは、PC で作業中の内容（開いているアプリケーションなど）を保持したまま、PC の動作を止める機能です。
なお、スリープ中は完全に電源が切れているわけではなく、ほぼ電力を使わない状態で電源が入っています。

MacBook シリーズの場合、蓋（ふた）を閉じることでスリープになります。

電源をオフにする（シャットダウンする）

画面左上にリンゴマークが描かれているアップルメニューをクリックします。
続いて、システム終了をクリックします。すると、PC の終了を行う処理が行われます。

画面が完全に消えたら電源が切れた状態です。
完全に終了しないうちにノート PC を閉じたり、収納したりするのは避けましょう。

再起動する

再起動とは、電源を一旦オフにしたあとにもう 1 回起動することです。新しいアプリケーションをインストールした後や、動作が不安定な時にリセットする目的などで使われます。

画面左上にリンゴマークが描かれているアップルメニューをクリックします。
続いて、再起動をクリックします。すると、PC の終了を行う処理が行われます。
シャットダウンと違うのは、電源が切れたあとに再び自動的に電源がオンになる点です。

PC の強制終了

PC の動作がおかしくなってしまった場合、強制終了しなければならないときがあります。いざというときに備え、手順を覚えておきましょう（いま試す必要はありません）。

まずは、control キー と option キーと command キーと 電源ボタン を同時に押します。
4つのキー・ボタンを同時押しすることになるので、ちょっと大変です。
上手くいくと、Mac の終了に関するメニューが現れるので、画面の指示に従って操作して終了してください。もし、保存してないデータがある場合は保存することもできます。

もし、上記の手順で終了に関するメニューが現れない場合は control キー と command キーと 電源ボタン を同時に押します。
この手順の場合、保存していないデータがあっても無視されます。

それでも動かない場合

この手順による強制終了はデータ破損など、トラブルの原因になることもあります。PC が操作を受け付けなくなるなど、やむを得ない場合のみに使用してください。

キーボードやマウスの操作をしても全く動かない場合、Mac の電源ボタンを長押しします。
電源が切れるまで長押しし続けてください。
