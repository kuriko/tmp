ダウンロードとは？

ダウンロード（download）とは、ネット上に存在しているファイルを、自分のパソコンにコピー・保存することを指します。
DL などと略されることもあります。また、俗語として「（データを）落とす」などと表現されることもあります。

ダウンロードとインストールの違い

ダウンロードとインストールはよく似た場面で使われる用語ですが、意味が異なります。

ダウンロード はネット上に存在しているファイルを、自分のパソコンにコピー・保存することです。
インストール はアプリをパソコンで使えるように設定することです。

家電の購入に例えてみましょう。
あなたは洗濯機を通販で注文したとします。洗濯機があなたの家に配達されるところまでが「ダウンロード」のようなものです。

しかし、洗濯機のような家電は届いたままの状態では使えません。コンセントや水道、排水管に接続するといった設置工事をする必要があります。これが「インストール」のようなものです。
インストール（設置工事）をしてくれる、専門知識のあるスタッフのようなものが インストーラー（インストール専用のアプリ）です。

もちろん、通販で購入できる商品の中には設置工事をせずに使えるものもあります。例えば本は家電と異なり、届いたらそのまま読めますし、服もそのまま着られます。
ダウンロードでも同様です。文章ファイルや画像ファイル、動画ファイルなどはアプリと異なり、ダウンロードしたらそのまま開くことができます。

実際にデータをダウンロードしてみよう

今回は N 予備校が用意した練習用のデータをダウンロードしてみましょう。

パソコンで Chrome を起動し、アドレスバーに https://git.io/renshu と入力して Enter キーを押します。
すると、N 予備校が用意した練習用のページが表示されます。

ダウンロードの練習 をクリックします。

ファイルをダウンロード をクリックします。

Chrome の画面下部にダウンロードの進行状況が表示されます。
残り時間の表示が消えたらダウンロード完了です。

このままファイルを開くこともできるのですが、まずこのファイルがどこに保存されているかを確認しましょう。 ファイル名の右側に表示されている V 字のボタンをクリックし、フォルダを開く をクリックします。

ダウンロード というフォルダが表示されます。これは「ドキュメント」フォルダなどのように、Windows が用意しているフォルダです。

練習用のファイルが見つけられたでしょうか。
練習用のファイルは ZIP 形式で圧縮されているので、展開しましょう。練習用のファイルを 右クリック し、すべて展開 をクリックします。
（ZIP で圧縮されたままでファイルを操作するとトラブルの原因となるので、必ず展開しましょう。）

無事に ZIP ファイルの中身が表示されたでしょうか。

ダウンロードしたファイルを整理する

今回はダウンロードフォルダでファイルを確認して展開しました。
しかし、ダウンロードフォルダにデータを置きっぱなしにすると、必要なファイルを見つけるのが大変になっていきます。

届いた宅配便がすべて玄関に積み上がっているような状態になってしまいます。

そのため、ファイル操作の回で学んだように、フォルダを作成して整理することをおすすめします。
「ダウンロード」フォルダに入ったデータは「ドキュメント」などに移動できます。

たとえば学生の方は授業別のフォルダを、働いている方はプロジェクトや取引先別などのフォルダを作ってダウンロードしたファイルを整理すると、あとから必要なファイルを見つけやすくなるでしょう。

Chrome でダウンロード時にフォルダを選択する

Chrome の初期設定では、ダウンロードしたファイルはすべて「ダウンロード」フォルダに保存されるようになっています。
しかし、フォルダを分けて整理する場合は毎回ダウンロードフォルダを開いて移動する手間が生じてしまい、ちょっと不便です。

Chrome には、ダウンロードするたびに保存するフォルダを選べる機能があります。
この手順はパソコンを使う上で必須ではありませんが、きちんと整理したい場合には便利機能なので設定してみましょう。

まず、Chrome の設定画面を開きましょう。Chrome の右上にある点が 3 つ並んだボタンをクリックし、設定 をクリックします。

続いて、画面左側にある 詳細設定 をクリックし、表示された ダウンロード をクリックします。

通常オフになっている ダウンロード前に各ファイルの保存場所を確認する を、クリックしてオンにします。

これで次回から、ファイルをダウンロードするときに保存するフォルダを指定する画面が表示されるようになります。

Tips　右クリックでダウンロード

右クリックでファイルをダウンロードすることもできます。保存したいコンテンツの上で右クリックすると、メニュー内に リンク先を別名で保存... という項目が表示されます。

リンク先を別名で保存...

これをクリックすると、保存先を指定するためのエクスプローラーの画面が表示されます。

保存先を選択する

「保存」をクリックすると実際にダウンロードが開始します。

アップロードとは？

アップロード（upload）とは逆に、自分のパソコンの中にあるファイルを、Web サイトに送信することです。
SNS に画像を添付する際、オンライン授業の課題を提出する際など様々な場面で使います。

実際にアップロードしてみよう

N 予備校で用意した練習用のサイトを使ってアップロードを体験してみましょう。

パソコンで Chrome を起動し、アドレスバーに https://git.io/renshu と入力して Enter キーを押します。
すると、N 予備校が用意した練習用のページが表示されます。

アップロードの練習 をクリックします。

ファイルの選択 ボタンをクリックします。

メモ帳などでファイルを開く時と同じ画面が表示されます。
アップロードするファイルをクリックし、「開く」をクリックします。今回は練習なのでどのようなファイルでもよいですが、ファイル操作の練習をした時にメモ帳で作成したファイルを選択してみましょう。

その後、送信ボタンをクリックします。

なお、今回の練習用 Web ページでは送信されたデータを収集していませんのでご安心ください。

アップロードの注意事項

一度アップロード（送信）してしまったファイルは基本的には取り消せないため、間違ったファイルをアップロードしないように注意しましょう。
また、他人の著作権などを侵害するファイルをアップロードすると、罪に問われることがあります。
