Wi-Fi ルーターの種類

「Wi-Fi ルーター」「無線 LAN ルーター」と呼ばれるものには、いくつか種類があります。
種類別の特徴を学び、自分の場合はどれを使うと良いのかを考えてみましょう。

  • 家庭用 Wi-Fi ルーター（有線ネット回線に接続するタイプ）
  • ポケット Wi-Fi ルーター
  • 家庭用 Wi-Fi ルーター（工事不要タイプ）

また、スマホを使った「テザリング」についても解説します。

家庭用 Wi-Fi ルーター（有線ネット回線に接続するタイプ）

「ルーター」の最も基本的な形です。

メリット

自宅に「光回線」「ケーブルテレビ（CATV）」など有線のネット回線があるなら、1 度このタイプのルーターを買ってしまえば追加で費用を支払う必要はありません。
また、通信が最も安定しているほか、通信制限（いわゆる「ギガが足りない」状態）が発生しないのも大きなメリットです。

パワフルなため、家の中の広い範囲で使用できます。

Wi-Fi だけではなく有線 LAN をサポートしている機種も多いため、オンラインゲームなどで有線 LAN が必要な場合にも役立ちます。

デメリット

有線のネット回線が必要になります。家に有線の回線がない場合、工事が必要になります。

ただし、マンションやアパートなどであれば、有線のネット回線がすでに設置されているところも多いです。コンセントの近くにネット用の端子がある場合も多いので、一度確認してみましょう。
自分の家で有線のネット回線が使えるかわからない場合、管理会社などに問い合わせてみるのもよいでしょう。

モバイルルーター（ポケット Wi-Fi ルーター）

先ほどの家庭用 Wi-Fi ルーターから、有線のネット回線を携帯電話回線に置き換えたものです。
また、「ポケット」といわれる名前の通り、コンパクトかつ充電式で、持ち運びできるのがメリットです。
略して「ポケファイ」などと呼ばれることもあります。

メリット

持ち運びできるため、家でも外出先でも同じ Wi-Fi を利用できます。
また、有線のネット回線がない場所でも使用できます。

デメリット

スマホのように月額料金が掛かります。また、携帯電話の回線を利用しているため、スマホと同様に通信制限が発生するものもあります。
特にパソコンは通信するデータのサイズがスマホより大きいため、通信制限の量はスマホよりも大きく見積もっておく必要があります。

また、小型で持ち運びを前提としているため、電波の到達範囲が狭いです。電波が届くのはポケットWi-Fiルーターの周囲だけです。
家中の機器に電波を届けることは難しいといえるでしょう。

通信が安定しない場合も多く、オンラインゲームの対戦などは難しい場合もあります。

持ち運びはできません（契約した住所以外で接続すると、制限が掛かる製品もあります）。

デザリング と書かれることがありますが、これは 間違い です。最初のテには濁点（゛）が付かない「テザリング」が正解です。

自分に合った Wi-Fi を選ぼう

以上のように、 3 種類のルーターを学びました。
使い方や家の状況などによって異なるため、どれがベストであると一概には言えませんが、それぞれの特徴を見て自分に合った Wi-Fi ルーターを選びましょう。

自宅でしかネットを利用しない場合

「会社や学校にはネット回線があり、家でネット回線が必要になるのはオンライン授業やテレワークの時だけ」などがこれに当てはまります。

こういった場合、まずは 有線のネット回線に接続するタイプのルーター を考えてみると良いでしょう。
テレビ会議やオンライン授業で通信量が増えても安定して通信できるためです。

自宅にすでに有線のネット回線が通っている場合、一度ルーターを買ってしまえば良いので、追加の出費も少ないです。

もし、自宅に有線の回線がない場合、有線の回線工事（光回線）をするか 工事不要のホームルーター タイプを検討するとよいでしょう。
オンラインゲームなどで回線の速度や安定性を重視するのであれば有線が必要ですが、パソコンをあまり使わない場合はホームルータータイプでも十分でしょう。

自宅に加えて、外出先でもネットを利用する場合

「家だけでなく、移動中や外出先のカフェなどでもパソコンでネットを利用したい」場合などがこれに当てはまります。

この場合、モバイルルーター を考えてみるのもよいでしょう。
ただし、最近のカフェなどは無料で利用できる Wi-Fi が設置されているところも多いため、モバイルルーターを買わなくても外出先でネットができる可能性もあります。

コラム: Wi-Fi と無線 LAN の違い

パソコンやスマホを無線でネットワークに接続するものを 無線 LANラン と言ったり、Wi-Fi と言ったりします。両者に違いはあるのでしょうか？

実は、厳密には違いがあります。
無線 LAN が登場したのは1990年代後半です。当時の無線 LAN はメーカーによって仕組みが微妙に異なり、異なるメーカー同士の製品は接続できないという問題がありました。
その問題を解決するためにメーカーによって作られた団体が「Wi-Fi アライアンス」です。Wi-Fi アライアンスは、メーカーが異なっても相互に接続できるかを確認するテストを定め、そのテストに合格した商品に「認証済み」（Wi-Fi CERTIFIED）のロゴを与えるようになりました。

つまり、他のメーカーの製品と接続できると確認済みの無線 LAN が「Wi-Fi」ということになります。
最近はそのテストに合格していない製品が販売されることは少ないため、「無線 LAN」と「Wi-Fi」がほとんど同じ意味として使われていると言えます。
