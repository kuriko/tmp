インストールするアプリ

Google Chrome（グーグル・クローム）というアプリをパソコンにインストールして使ってみます。
Google Chrome はウェブブラウザ（Web Browser）とよばれる、ネットを閲覧するためのアプリの一種です。

Mac には標準で Safari（サファリ）と呼ばれるブラウザが導入されていますが、この講座ではこれ以降 Google Chrome を使います。
なぜなら、Chrome は世界で一番使われている（ユーザが多い）ブラウザであり、世の中の Web サイトのほとんどが Google Chrome に対応するように作られているからです。また、この後の章で登場する Google の Web サービスとも相性が良いのです。

なお、Safari が使えなくなるわけではありません。もし Safari が必要になったときは、引き続き Safari を利用することもできるのでご安心ください。

Chrome インストーラーのダウンロード

まず、標準のブラウザである Safari を起動します。
画面下部の Dock から Safari をクリックしてください。

Dock にある Safari をクリックする

URL（アドレス）を入力する欄に半角英数字で Chrome と入力して Enter を押し、検索エンジンの Google にアクセスしましょう

検索欄に Chrome と入力し Enter を押す

すると、検索結果が表示されます。
google.co.jp や 公式サイト などと表示されている検索結果をクリックします。

信頼できるサイトにアクセスする

以下のような Google Chrome 公式サイトが表示されます。
画面の中央にある Chrome をダウンロード というボタンをクリックします。

Chrome をダウンロードをクリック

ダウンロードを許可するかの確認画面が出てきた場合は、許可を押してください。

許可をクリック

ダウンロードが開始されると、右上にダウンロードアイコンが表示されます。これをクリックすることでダウンロードの進捗がわかります。

Safari はまだ使うので、開いたままにしてください。

インストーラーの実行

Safari の右上にあるダウンロードアイコンをクリックして、ダウンロードが完了したことを確認します。完了していたら、ダウンロードしたファイル名をクリックします。

ダウンロードしたファイルをクリック

ダウンロードしたファイルを展開する画面がでた場合は、何もせずに待ちます。

この時点で Safari の画面は閉じて構いません。 左上の ✕ ボタンを押してウィンドウを閉じてください。

ファイルを展開している画面

展開が終わると、インストーラの画面が開きます。
上にある Chrome のアイコンを、下のアプリケーションフォルダのアイコンに、ドラッグアンドドロップします。

上のアイコンを下のアイコンにドラッグアンドドロップ

これでインストールが完了しました。

初回起動時の設定

インストールした Chrome を起動してみましょう。

Command + Space を押して、 Spotlight 検索を開きます。

検索欄に Chrome と入力すると、 Google Chrome のアイコンが表示されるので、 Enter を押して起動してください。

Spotlight の画面

Chrome が起動すると、右上に通知を許可するかのダイアログが出てきます。 特にこだわりがない人は許可を押してください。

許可をクリック

Google Chrome へようこそという画面が開きます。
Chrome を使い始める前の設定を聞かれます。

特にこだわりがない人は、「使用統計データや障害レポートを Google に自動送信して Google Chrome の機能向上に役立てる」のチェックを外し、「Google Chrome を起動」ボタンをクリックします。

Chrome を使い始める前の設定

「Google Chrome を既定のブラウザとして設定する」を選択した人はもう一度確認の画面が出てくるので、「"Chrome"に変更」ボタンをクリックします。

Chrome を使い始める前の設定

ここまで済むと、Chrome を使い始める際に、便利にするための手順が表示されます。 開始する をクリックします。

起動直後の画面

Google が提供するアプリをブックマーク（お気に入り）に登録する手順が表示されますが、特にこだわりがなければ「スキップ」をクリックしてください。

ブックマークのカスタマイズ画面

Chrome の背景デザインを選ぶ画面が表示されます。好きなものに変更できます。
特にこだわりがなければ「スキップ」をクリックしても構いません。

デザインのカスタマイズ画面

Google アカウントで Chrome にログインするかどうかを聞かれる画面が表示されます。特に必要性はないので、「いいえ」ボタンをクリックします。
もし Google のアカウント（ID）をお持ちであれば、ログインしても問題ありません。

Chrome にログインするか尋ねる画面

最後に、Dock に Google Chrome を追加します。

Dock にある Google Chrome のアイコンを右クリックし、オプションにある「Dock に追加」をクリックします。

Dockから Chrome を起動できる

これで Dock に Google Chrome が追加されました。 今後は、Dock に追加されたアイコンをクリックすることで、Google Chrome を起動させることができます。

以上で、Google Chrome の初期設定は完了です。
