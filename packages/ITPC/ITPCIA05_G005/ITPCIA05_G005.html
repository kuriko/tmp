LINE をインストールして使ってみよう

多くの人がスマホで使っている LINE ですが、実はパソコンでも使用できます。 今回は Windows のパソコンに LINE をインストールし、使ってみましょう。

  • この節を学ぶと出来るようになること
  • LINE のダウンロード・インストール
  • LINE をパソコンから使ってみよう

この節を学ぶと出来るようになること

  • LINE のインストール方法を学べます。LINE のインストーラーは Windows 用アプリのインストーラーとしてかなり典型的なものであるため、他のアプリをインストールするときにも参考になります。
  • パソコン版の LINE を使えるようになります。パソコンで作成したファイルを他の人に送る場合に便利です。
  • パソコンで LINE を使えると、普段のメッセージのやり取りがタイピング練習になります。

LINE のダウンロード・インストール

Chrome で LINE の公式サイトにアクセスしましょう。アドレスバーに line.me と入力するか、Google 検索で「LINE」と検索しましょう。

公式サイト左上にある「ダウンロード」のリンクをクリックします。

Windows 版のダウンロードを行います。「Windows版をダウンロード」をクリックします。

Chrome の左下にダウンロード状況が表示されます。

ダウンロードが完了し、緑色のアイコンが現れたらこれをクリックしましょう。

言語を選ぶ画面が表示されます。日本語 になっていることを確認して OK をクリックします。

次へ をクリックします。

利用規約に同意したら 同意する をクリックします。

インストールが終了するまで待ちます。

閉じる をクリックします。

LINE をパソコンから使ってみよう

インストーラーを閉じると、LINE が自動的に起動します。

メールアドレス（電話番号）とパスワードを覚えている場合、それでログインしても大丈夫です。
しかし、スマホのLINEから QR コードを使うことでより便利にログインできます。

スマホでの操作

ここからは少しだけスマホを使います。

自分のスマホのLINE で QR コードリーダーを開きます。友だち追加をするときに使うものと同じです。

パソコンの画面に表示されている QR コードを読み取ります。

ログイン をタップします。

他の人のパソコンから勝手にログインされてしまうのを防ぐために、初回は本人確認が行われる場合があります。
パソコンの LINE の画面に表示された数字をスマホに入力します。

パソコンでの操作

スマホを使い、パソコンから LINE にログインできました。

基本的な使い方や機能はスマホ版と同じですが、パソコンで作成したファイルを添付して友達に送れるといったメリットがあります。また、普段からパソコンで LINE を使うと、キーボード入力の練習にもなるでしょう。

次回以降はデスクトップに作成された LINE をダブルクリックして起動しましょう。
