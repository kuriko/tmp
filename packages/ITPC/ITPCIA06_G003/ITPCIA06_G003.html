Google Chrome を起動しよう

Google Chrome をインストールすると、Launchpad にアイコンが現れます。これをダブルクリックしましょう。

Chrome のアイコン

Google Chrome の基本的な使い方（アドレスバーとタブバー）

Chrome の画面上部に入力欄があります。これをアドレスバーといいます。
ここに Web サイトのアドレス（URL）を入力すると、そのサイトに移動できます。また、単語などアドレス以外のものを入力すると Google 検索が開きます。

例えば、ここに「nnn.ed.jp」と半角英数字で入力し、Enter キーを押してみると、N 高等学校のサイトが表示されます。
このように、Web サイトに移動することを アクセスする といいます。

N 高等学校のページを開いた状態で、別のページも開いてみましょう。
Chrome などのブラウザには タブ と呼ばれる機能があります。タブは Web サイトを表示するための「枠」です。説明するよりも実際に使ってみるほうが早いので、早速試してみましょう。

Chrome の一番上（タブバー）に、さきほど開いたページのタイトル「N 高等学校」が表示されていると思います。この横に +（プラス）が表示されているので、これをクリックしてみましょう（また、Command + Tを同時押ししても同様です）。

すると、新しく初期の画面が表示されました。この状態を 新しいタブを開いた といいます。

ここで再び、アドレスバーに入力してみましょう。今回は「nnn.ed.nico」と入力し、Enter キーを押してみましょう。
今度は N 予備校のサイトが表示されるはずです。

タブバーに表示されている Web サイトのタイトルをクリックすることにより、タブを切り替えられます。

このように、タブを活用すると複数の Web サイトを同時に開けます。

タブを閉じる

それぞれのタブはタブ名の右側にある × 印をクリックすると閉じられます。

もし、間違ってタブを閉じてしまった際は、タブバーを右クリックし、 閉じたタブを開く をクリックすることで直前に閉じたタブを再び開けます（Command + Shift + T を同時押ししても同様です）。

新しいタブで開く

Web サイトの中のリンクは通常、同じタブで開きます。

しかし、リンクを右クリックして 新しいタブを開く をクリックすると、新しいタブでページを開けます（Command キーを押しながらリンクをクリックしても同様です）。

なお、Web サイト側の設定により、この操作をしなくても自動的に新しいタブで開くリンクもあります。

ページ内検索

Chrome には、ページ内にある単語や文字が含まれているかどうかを検索する機能があります。

Chrome の画面右上にある、点が 3 つ描かれているボタンをクリックし、検索 をクリックします。

Chrome の右上に小さな入力欄が現れます。これがページ内検索欄です。
ここに入力した文字がページ内にあると、強調表示されます。
ページ内の複数箇所に検索ワードに一致する場所があるときは、入力欄の右側にある上下のマークをクリックすることで、次の出現場所・前の出現場所に移動できます。

検索を終わるときは検索欄の右側にある × ボタンをクリックしましょう。

ブックマークを活用する

ブックマークはよくアクセスするページや、後から読みたいページなどを登録しておく機能です。
アドレスバー右端にある星マークをクリックし、完了 をクリックすると現在表示しているページをブックマーク登録できます。

今回は N 高等学校のサイト（nnn.ed.jp）をブックマーク登録してみましょう。

ブックマークを表示する

ブックマークに登録したページは ブックマークバー に表示されます。これをクリックするとアクセスできます。

ブックマークバーが表示されていないときは Command + Shift + B を同時押しすると表示・非表示を切り替えできます。

ブックマークを整理する

登録したブックマークの数が増えてきたら、ブックマークを整理するとよいです。その方法を覚えておきましょう。
Chrome の画面右上にある、点が 3 つ描かれているボタンをクリックします。

ブックマーク から ブックマークマネージャ をクリックします。

ブックマークマネージャでは、登録済みのブックマークを削除したりフォルダに入れたりして管理できます。

今回はフォルダを作成してみましょう。青い背景のところにある、点が 3 つ並んでいるボタンをクリックします。

新しいフォルダを作成をクリックします。

フォルダ名を入力します。ここでは「N 高」と入力し、保存 をクリックします。

先ほどブックマークに登録したサイト（ここでは「N 高等学校」のサイト）を、「N 高」フォルダにドラッグ・アンド・ドロップします。

ブックマークの移動に成功すると、「N 高」フォルダが残されるはずです。「N 高」フォルダをダブルクリックしてみましょう。

中身が「N 高等学校」のサイトになっているはずです。
これが確認できたら新しいタブを開いてみましょう。

無事にブックマークバーでも「N 高」フォルダの中に「N 高等学校」のサイトが登録されているはずです。

シークレットウィンドウ

一時的に人のパソコンを借りるときなど、ID やパスワードなどの入力履歴や閲覧履歴を残したくない場面があります。
そういった場合はシークレットモードを活用しましょう。

画面右上のボタンをクリックし、シークレットウィンドウを開く をクリックします。

すると、新しいウィンドウが表示されます。このウィンドウ内での閲覧履歴や入力履歴は使用しているパソコンには保存されません。
ただし、Web サイトや通信会社、ネットワークの管理者側にはアクセスした履歴が残るため、ネット上で完全な匿名になるというわけではありません。

まとめ

  • アドレスバーに検索単語やURLを入力すると、ページを移動できる。
  • ページ内検索機能で、表示しているページ内にある文字を検索できる。
  • よく見るページは「ブックマーク」できる。
    • ブックマークはフォルダで管理できる。
  • 閲覧履歴や入力履歴を残したくないときは、シークレットウィンドウを使うとパソコン内に記録が残らない。
