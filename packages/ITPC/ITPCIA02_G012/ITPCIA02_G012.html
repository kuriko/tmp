コピー・アンド・ペーストをしよう

文字や画像などをコピーして、別の場所に貼り付けることを コピー・アンド・ペースト と呼びます。略して「コピペ」と言われます。
コピーは「複製」、ペーストは「貼り付け」という意味です。

今回の目標

  • 文章（文字）をコピー・アンド・ペーストする手順を確認する
  • コピー・アンド・ペーストに関連する便利な機能を知る
  • 著作権上の注意事項を確認する

文章のコピー・アンド・ペースト

メモ帳を起動しましょう。
「コピペの練習」と入力してください。

メモ帳に入力する

マウスで「コピペ」の部分だけドラッグします。これで「コピペ」という文字が選択された状態になります。

「コピペ」という文字を選択
コラム: キーボードを使って微調整

マウスのドラッグ操作で文字を選ぶとき、狙ったところを上手く選択できないこともあります。たとえば、今回の例なら「コピ」だったり「コピペの」などを選択してしまう場合があるでしょう。
そういったときはキーボードを使って微調整できます。
Shift キーを押しながら、矢印キー（←, →）を押すことで文字の選択範囲を 1 文字ずつ調整できます。

選択された部分を右クリックして コピー をクリックします。

「コピー」をクリック

これで コピー されました。続いて、別の場所に ペースト しましょう。
「コピペの練習」という文の後に、「をします。」と入力しましょう。

メモ帳に入力する

「をします。」と入力し終えたら、メモ帳の画面内を 右クリック して 貼り付け をクリックします。

「貼り付け」をクリック

すると、カーソルがある場所に貼り付けされます。

文字が貼り付けられた

これでコピペの基本操作が出来ました。

なお、一度コピーをした後に再びコピーを行うと、前にコピーした内容は消えてしまいます。コピーした内容を保持している場所は クリップボード とも呼ばれます。

「すべて選択」と「切り取り」

応用編として「すべて選択」と「切り取り（カット）」もやってみましょう。

「コピペの練習をします。コピペ」となっているはずですが、この文を全部選択してみましょう。
短い文章であればドラッグ操作でも可能ですが、画面幅を超えるような文章では難しくなってくるので、「すべて選択」を覚えておくととても楽になります。

メモ帳の画面上で 右クリック して すべて選択 をクリックします。

「すべて選択」をクリック

メモ帳画面内の文字がすべて選択されました。
この状態で再び 右クリック し、 切り取り をクリックします。

「切り取り」をクリック

すると、文が消えるはずです。
ここで「切り取りのテスト。」と入力しましょう。

メモ帳に入力する

入力したら、 右クリック して 貼り付け をクリックします。

「貼り付け」をクリック

先ほど切り取りした文が貼り付けられたと思います。

「コピー」はもともとあった文を削除せずに「貼り付け」できるようにします。
一方、「切り取り」はもともとあった文を削除して「貼り付け」できるようにします。
状況によって使い分けましょう。

ショートカットキー

コピー・アンド・ペーストでは、ショートカットキーを使えます。

ショートカットキーとは、マウス操作と同じ機能をキーボードの操作で行うものです。
必ず覚える必要はありませんが、活用できると便利です。

コピー

コピーをするときは Ctrl キーを押しつつ C キーを追加で押します。
（以後、このような同時押しを Ctrl + C のように表記します）

実際にやってみましょう。

先ほどと同様に、メモ帳で文字を選択します。

文字を選択する

この状態で右クリックをせず、Ctrl + C を押します。

貼り付け（ペースト）

貼り付けのショートカットキーは Ctrl + V です。

これもコピーと同様に、実際に試してみましょう。

すべて選択

Ctrl + A

切り取り

Ctrl + X

様々なコピペ

今回はメモ帳で文字のコピー・アンド・ペーストを使いましたが、様々なアプリでこの機能は搭載されています。
他のアプリにデータをコピー・アンド・ペーストできるものもあるので、いろいろ試してみるとよいでしょう。

著作権には要注意

他の人によって作られた文章や画像、ファイルなどには 著作権 があります。これを侵害した場合、著作者から訴えられ、罪に問われてしまうことがあります。
また、学校のレポートなどで他の人の文章をコピペした場合、基本的には 不正行為 としてみなされます。

パソコンの機能でコピー・アンド・ペーストできるからといって、文章や画像などを自由に使って良いわけではないことに気をつけましょう。

参考：著作権についての詳細は みんなの著作権教室 といったサイトをご覧ください。

まとめ

  • 文字を選択して右クリックし、「コピー」をクリックするとコピーできる。
  • 貼り付けたい先で右クリックし、「貼り付け」をクリックするとコピーした文字を貼り付けできる。
  • コピーの場合、貼り付けても元の文字は残る。切り取り（カット）の場合、貼り付けると元の文字は消える。
  • コピー・アンド・ペーストには便利なショートカットキーがある。
    • コピー：Ctrl + C
    • 切り取り（カット）：Ctrl + X
    • 貼り付け（ペースト）：Ctrl + V
  • アプリによっては文字以外もコピー・アンド・ペーストできるので、色々試してみるとよい。
  • 他の人の文章や画像などをコピー・アンド・ペーストするときは、著作権を侵害しないように気をつける。
