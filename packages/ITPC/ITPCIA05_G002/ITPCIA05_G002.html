インストールするアプリ

Google Chrome（グーグル・クローム）というアプリをパソコンにインストールして使ってみます。
Google Chrome はウェブブラウザ（Web Browser）とよばれる、ネットを閲覧するためのアプリの一種です。

Windows には標準で Microsoft Edge（マイクロソフト・エッジ）と呼ばれるブラウザが導入されていますが、この講座ではこれ以降 Google Chrome を使います。
なぜなら、Chrome は世界で一番使われている（ユーザが多い）ブラウザであり、世の中の Web サイトのほとんどが Google Chrome に対応するように作られているからです。また、この後の章で登場する Google の Web サービスとも相性が良いのです。

なお、Edge が使えなくなるわけではありません。もし Edge が必要になったときは、引き続き Edge を利用することもできるのでご安心ください。

Chrome インストーラーのダウンロード

まず、標準のブラウザである Edge を起動します。
スタートメニューのアプリ一覧をスクロールし、Microsoft Edge という名前のアプリをクリックしてください。

Microsoft Edge をクリックして起動

URL（アドレス）を入力する欄に半角英数字で Chrome と入力して Enter を押し、検索エンジンの Google にアクセスしましょう

Edge でのURL入力欄

すると、検索結果が表示されます。
google.co.jp や 公式サイト などと表示されている検索結果をクリックします。

検索結果をクリック

以下のような Google Chrome 公式サイトが表示されます。
使用統計データと障害レポートを Google に自動送信して Google Chrome の機能向上に役立てるのチェックを外し、 画面の中央にある Chrome をダウンロード というボタンをクリックします。

Chrome をダウンロードをクリック

ダウンロードが完了すると、自動でページが切り替わり、画面下部にはダウンロードしたファイル名が表示されます。
次でも使うため、まだこの画面は閉じないで下さい。

ダウンロードが完了した画面

インストーラーの実行

画面下部に出てきた ChromeSetup.exe の ファイルを開く をクリックして、インストーラーを実行してください。

ファイルを開くをクリック

ユーザーアカウント制御というウィンドウが出てきたときは はい をクリックしてください。

ユーザーアカウント制御の画面

ファイルのダウンロード作業などが行われるので待ちましょう。

この時点で Microsoft Edge の画面は閉じて構いません。 右上の ✕ ボタンを押してウィンドウを閉じてください。

インストール中の画面

完了したら、Google Chrome が起動します。

初回起動時の設定

Chrome を使い始める際に、便利にするための手順が表示されます。 開始する をクリックします。

起動直後の画面

Google が提供するアプリをブックマーク（お気に入り）に登録する手順が表示されますが、特にこだわりがなければ「スキップ」をクリックしてください。

ブックマークのカスタマイズ画面

Chrome の背景デザインを選ぶ画面が表示されます。好きなものに変更できます。
特にこだわりがなければ「スキップ」をクリックしても構いません。

デザインのカスタマイズ画面

Chrome をデフォルトのブラウザ（Windows が標準で使うブラウザ）にするか尋ねる画面が表示されます。 Web プログラミングにおいては Chrome をデフォルトのブラウザに設定しておく方が便利なので、設定しておきましょう。「デフォルトとして設定」をクリックします。

既定のブラウザにするか尋ねる画面

すると、デフォルトのアプリを変更する Windows の設定画面が表示されます。
下の方に「Web ブラウザ」という欄があるので、ここに表示されている現在のブラウザ（Microsoft Edge や Internet Explorer）の名前をクリックし、表示されたアプリ一覧から Google Chrome をクリックします。

既定のアプリを設定する画面

以下の画面が出てきた場合は、「強制的に変更する」をクリックします。
無事に設定ができたら、「デフォルトのアプリ」画面は閉じてください。

「強制的に変更する」をクリック

Chrome に戻ると、Google アカウントで Chrome にログインするかどうかを聞かれる画面が表示されます。特に必要性はないので、「いいえ」ボタンをクリックします。
もし Google のアカウント（ID）をお持ちであれば、ログインしても問題ありません。

Chrome にログインするか尋ねる画面

今後は、デスクトップに追加された「Google Chrome」のアイコンをダブルクリックすることで、Chrome を起動できます。

デスクトップから Chrome を起動できる

以上で、Google Chrome の初期設定は完了です。

まとめ

  • インストーラータイプのアプリは、まずインストーラーをダウンロードする
  • ダウンロードしたインストーラーを実行すると、インストールできる
  • アプリによっては初回起動時に設定が必要なものがある
