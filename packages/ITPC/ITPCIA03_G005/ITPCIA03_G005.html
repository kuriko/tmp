キーボードとは？

スマホは フリック入力 など、画面上に現れるもので操作しますが、パソコンの場合はボタンがたくさん並んだ キーボード を使います。
キーボード上のボタンをキーと呼びます。

キーボードの配列

キーボード上のキーの並び方を配列といいます。配列は主に言語（日本語、韓国語、中国語、フランス語……）によって異なります。

日本語用キーボードの基本的な配列は決まりがあるのですが、パソコンの製品によって若干異なる場合があります。
この教材では基本的な JIS配列（ジス配列）と呼ばれる配列を前提として解説を進めていきます。

JIS 配列をベースにした Windows ノートパソコンの場合、だいたい以下のような配列になっているはずです。

Windows のキーボード（例）

最初に、よく使うキーの名称と役割を学びましょう。
もちろん、実際に使ってみないと分からないと思うので、ここではキーの名前と場所を確認できれば十分です。

Enter （エンター）

改行したり、変換を確定したりします。

Backspace （バックスペース）

1つ前の文字を削除します。パソコンによっては「BS」と略されている場合があります。

Delete （デリート）

1つ後の文字を削除したり、選択した図形やファイルなどを削除します。パソコンによっては「Del」と略されている場合があります。

Shift（シフト）

大文字や記号を入力する場合に使います。パソコンによっては、左側・右側の両方に存在する場合がありますが、基本的にはどちらも同じ役割です。自分が押しやすい方を使いましょう。

注意が必要なキー

いくつかのキーは間違って押してしまうことで、意図しない動作をすることがあります。
ただし、これらはもう1回押せばすぐに戻せるため、あわてたり怖がったりする必要はありません。

CapsLock（キャップス・ロック）

CapsLock キーをオンにすると、入力するアルファベットがすべて大文字になります。
Shift キーと CapsLock キーを同時に押すことでオン・オフが切り替えられます。
アルファベット文字が全て大文字になってしまう……というときは、誤って CapsLock がオンになっている可能性があります。もう 1 回 Shift キーと CapsLock キーを同時に押してオフにしてください。

また、パソコンによっては CapsLock キーをオンにすると、CapsLock キーのランプが光るものもあります。

Insert（インサート）

Insert は「挿入」を意味する英単語で、文字の「挿入モード」と「上書きモード」を切り替えるキーです。
パソコンの入力は通常、「挿入モード」になっているのですが、これを押すと「上書きモード」になります。上書きモードになると、既に入力されている文字が消されながら（上書きされながら）入力されます。
既に入力した文字が消えてしまう……というときは、誤って上書きモードがオンになっている可能性があります。もう 1 回 Insert キーを押して挿入モードに切り替えましょう。

NumLock（ナムロック）

NumLock は「テンキー」と呼ばれる、数字専用のキーをオンにするための機能です。
デスクトップパソコン用のキーボードでは、右側にテンキーが存在しているものがあります。NumLock キーを押すと、これが有効になり、効率的に数字を入力できるようになります。

デスクトップパソコンのテンキー

しかし、多くのノートパソコンでは、キーボードの場所を節約するために通常の文字キーにテンキー機能を持たせています。ノートパソコンで NumLock をオンにするとこれを使えるようになるのですが、代わりに通常の文字が入力できなくなってしまいます。
普通の文字を入力しているはずなのに数字になってしまった……というときは、誤って NumLock をオンにしてしまっている可能性があります。もう 1 回 NumLock キーを押してオフにしてください。

コラム:　なぜアルファベット順に並んでいないの？

キーボード上のアルファベットは A,B,C,D,E,F... ではなく、 Q,W,E,R,T,Y... のように並んでいます。これを QWERTY配列（クワーティはいれつ）と呼びます。
元々、英語圏を中心に使われており、皆さんが使っている日本語キーボードの JIS 配列もこれを日本語用に発展させたものです。

QWERTY 配列は一見すると不便に見えます。アルファベット順に並んでいる方が分かりやすいように思いますよね。

この配列の起源は 1800 年代までさかのぼります。
当時、キーを押すと文字が印字される「タイプライター」とよばれる機械がありました。PC のようにアプリが動いたり計算ができたりする物ではなく、ただ単に文字を打ち込むだけの機械です。

タイプライター

当時、仕事でタイプライターを使っていた人として代表的なのが電報のオペレーターたちでした。モールス信号を聞いて、それを文字（英語）にする仕事をするオペレーターたちにとって、素早く文字を打ち込める配列が重要視されていました。
現場のオペレーターたちから「このキーの位置が不便なので変更してほしい」「英語の単語では re や er がよく出てくるので、入力しやすいように並べてほしい」といった声が次々に届き、タイプライターのメーカーはそれを基にキー配列を変更していきました。
その結果として生まれたキーの配列に特許が申請され、他のメーカーは特許を侵害しないように新たなキー配列を生み出す、といったことが次々と行われた結果、QWERTY 配列が出来上がったと言われています。

  • 参考資料:パソコンのキーボードは，なぜABC順・五十音順ではないのですか （ことば研究館）

文字の消し方

文字を入力していると、間違えることもあります。ここでは、入力した文字の消し方を学びましょう。

BackSpace（バックスペース）というキーを使います。右上の方にあるはずです。
これを押すと、 1 つ前（左側）の文字を消す事ができます。

また、Delete（デリート）というキーもあります。これを押すと、1 つ後ろ（右側）の文字を消すことが出来ます。

なお、Delete キーは Windows と Mac で機能が少し違うので、両方の OS を使った経験がある人は間違えないように気をつけましょう。

数字やアルファベットを入力する

まずは練習として、数字やアルファベットを入力してみましょう。

その前に、パソコンにおける数字やアルファベットにはいくつか種類があります。それは全角（ぜんかく）と半角（はんかく）です。

全角と半角の例

見た目は似ていますが、パソコンは全角と半角を「別の文字」として認識します。そのため、間違ってしまうとエラーの原因になることがあります。
基本的に数字やアルファベットを入力する場合は半角で行いましょう。

Windows ではキーボード左上にある 半角/全角 キーを押すことで半角入力・全角入力を切り替えることができます。画面右下に「A」と表示される半角入力モード（直接入力モード）になっているかを確認してください。

半角入力モード（直接入力モード）

では、半角数字を入力してみましょう。

数字キーをファンクションキー（F1など）と間違えないように気をつけてください。

日本語を入力する

パソコンで日本語入力をする場合、「ひらがな入力」と「ローマ字入力」があります。
ひらがな入力はキーボードに描かれているひらがなを押して入力するもので、ローマ字入力はアルファベットを押してローマ字の形で入力するものです。

これからパソコンを習得する場合、ローマ字入力を強くおすすめします。
なぜなら、ローマ字入力の方が覚えるキーの数が少なくて済むからです。ひらがな入力をする場合、英語のアルファベット26文字に加え、46文字のひらがなキー、さらに記号を覚える必要があります。
ローマ字入力なら、アルファベット26文字と記号を覚えるだけで済みます。

また、ローマ字入力のほうが記号や数字を入力しやすいというメリットもあります。パソコンで学校の課題に取り組んだり、会社の仕事をしたり、プログラミングをする場合には、日本語だけではなく必ず記号や数字を入力することになります。
ひらがな入力の場合、切り替えの手間が増えてしまいます。

こういった理由から N 予備校でも基本的にはローマ字入力を使います。
もし、ローマ字入力に自信がない……という方は、付録の「ローマ字入力を覚えよう」のテキストをご覧ください。

ホームポジション

キーボードはどんな指でも押すことができますが、正しい入力姿勢が決まっています。これが「ホームポジション」です。
きちんとホームポジションで入力できるようになると、高速でミスの少ない入力が可能になります。また、キーに書かれている文字を見ずに、画面を見たまま入力する タッチタイピング も可能になります。

その他のキー

今回は操作しなかったキーについても少しだけ知っておきましょう。
機能まで覚えられない場合は、名前と場所だけ覚えておけば十分です。

Tab（タブ）

キーボードの一番左側にあります。

複数の入力欄があるときに、次の入力欄に移動します。
また、字下げの機能もあります。

Ctrl（コントロール）

キーボードの一番左下・右下にあります。

ショートカットキーの操作などに使います。ショートカットキーとは、本来はマウスでする操作をキーボードで素早く行うためのコマンドです。
パソコンによっては左右両方に存在する場合がありますが、基本的にはどちらも同じ役割です。

Alt（オルト）

キーボードの左下にあります。

ショートカットキーの操作などに使います。パソコンによっては左右両方に存在する場合がありますが、基本的にはどちらも同じ役割です。
英単語の Alternate（オルタネート）の略です。読み方は「アルト」ではないので注意しましょう。

F1 ～ F12（ファンクションキー）

キーボードの一番上の段にあります。

様々な機能が割り当てられたキーです。
パソコンによってはfnキーと同時に押すことで機能することもあります。

通常の数字キーとは違うので気をつけてください。

fn（エフエヌ）

キーボードの一番下の段にあります。

他のキーと同時に押してショートカットキーとして使います。
このキーで使える機能はパソコンによって異なります。

Windows（ウィンドウズ）

キーボードの左下にあります。四角が4つ描かれたロゴ（Windowsのロゴマーク）が記載されているキーです。

スタートメニューを開きます。他のキーと同時に押してショートカットキーとして使うこともあります。
