キーボードとは？

スマホは フリック入力 など、画面上に現れるもので操作します。しかしパソコンの場合は、ボタンがたくさん並んだ キーボード を使います。
キーボード上のボタンをキーと呼びます。

キーボードの配列

キーボード上のキーの並び方を配列（はいれつ）といいます。配列は主に言語（日本語、韓国語、中国語、フランス語……）によって異なります。

日本語用キーボードの基本的な配列は決まりがあるのですが、パソコンの製品によって若干異なることがあります。
この教材では基本的な JIS配列（ジス配列）と呼ばれる配列を前提として解説を進めます。

JIS 配列をベースにした Windows ノートパソコンの場合、だいたい以下の配列になっているはずです。

Windows のキーボード（例）

最初に、よく使うキーの名称と役割を学びましょう。
もちろん、実際に使ってみないと分からないと思うので、ここではキーの名前と場所を確認するだけで十分です。

Enter （エンター）

キーボードの右側の方にある大きなキーです。
改行したり、変換を確定したりします。

Backspace （バックスペース）

キーボードの右上の方にあります。
1つ前の文字を削除します。パソコンによっては BS などと略されている場合があります。

Delete （デリート）

キーボードの右上の方にあります。
1つ後の文字を削除したり、選択した図形やファイルなどを削除したりします。Del と略されている場合があります。

Shift（シフト）

キーボードの左下の方にあります。
大文字や記号を入力する場合に使います。パソコンによっては、左下・右下の両方にあったりしますが、基本的にはどちらも同じです。押しやすい方を使いましょう。

続いて、Windows アクセサリの中にある メモ帳 をクリックします。

以下のようなウィンドウが現れたら OK です。

起動したメモ帳

起動できたら文字を入力してみましょう。

文字の消し方

文字を入力していると、間違えることもあります。ここでは文字の消し方を学びましょう。

BackSpace（バックスペース）というキーを使います。右上の方にあるはずです。
これを押すと、 1 つ前（左側）の文字を消す事ができます。

また、Delete（デリート）というキーもあります。これを押すと、1 つ後ろ（右側）の文字を消すことが出来ます。

なお、Delete キーは Windows と Mac で機能が少し違うので、両方の OS を使った経験がある人は間違えないように気をつけましょう。

数字やアルファベットを入力する

まずは練習として、数字やアルファベットを入力してみましょう。

その前に知っておくべきことがあります。それは、パソコンには数字やアルファベットに種類があることです。
全角（ぜんかく）と半角（はんかく）があります。

全角と半角の例

見た目は似ていますが、パソコンは全角と半角を「別の文字」として認識します。そのため、間違ってしまうとエラーの原因になることがあります。
基本的に数字やアルファベットを入力する場合は半角で行いましょう。

Windows ではキーボード左上にある 半角/全角 キーを押すことで半角入力・全角入力を切り替えられます。画面右下に 「Ａ」 と表示される半角入力モード（直接入力モード）になっているかを確認してください。

半角入力モード（直接入力モード）

では、半角数字を入力してみましょう。

数字キーをファンクションキー（F1など）と間違えないように気をつけてください。

日本語を入力する

パソコンで日本語入力をする場合、「ひらがな入力」と「ローマ字入力」があります。
ひらがな入力はキーボードに描かれているひらがなを押して入力するもので、ローマ字入力はアルファベットを押してローマ字の形で入力するものです。

これからパソコンを習得する場合、ローマ字入力を強くおすすめします。
なぜなら、ローマ字入力の方が覚えるキーの数が少なくて済むからです。ひらがな入力をする場合、英語のアルファベット26文字に加え、46文字のひらがなキー、さらに記号を覚えなければなりません。
ローマ字入力なら、アルファベット26文字と記号を覚えるだけで済みます。

また、ローマ字入力のほうが記号や数字を入力しやすいというメリットもあります。パソコンで学校の課題に取り組んだり、会社の仕事をしたり、プログラミングをする場合には、日本語だけではなく必ず記号や数字を入力することになります。
ひらがな入力の場合、切り替えの手間が増えてしまいます。

こういった理由から N 予備校でも基本的にはローマ字入力を使います。
もし、ローマ字入力に自信がない……という方は、付録の「ローマ字入力を覚えよう」のテキストをご覧ください。

ホームポジション

キーボードはどんな指でも押すことができますが、正しい入力姿勢が決まっています。これが ホームポジション です。
きちんとホームポジションで入力できるようになると、高速でミスの少ない入力が可能になります。また、キーに書かれている文字を見ずに、画面を見たまま入力する タッチタイピング も可能になります。

左手の人指し指を F、右画の人差し指を J に置いた状態がホームポジションです。
以下の画像を参考に、正しい指で入力してみてください。

ホームポジション

一度、変な入力の癖がついてしまうと正しい入力姿勢を習得するのは大変です。出来る限り早めにホームポジションを習得するのが望ましいでしょう。
ただし、「ホームポジションで入力出来ないとパソコンを使ってはいけない」というわけではありません。気楽に少しずつ慣れていきましょう。
