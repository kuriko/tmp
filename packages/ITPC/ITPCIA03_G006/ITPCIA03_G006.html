キーボードとは？

スマホは フリック入力 など、画面上に現れるもので操作しますが、パソコンの場合はボタンがたくさん並んだ キーボード を使います。
キーボード上のボタンをキーと呼びます。

キーボードの配列

キーボード上のキーの並び方を配列といいます。配列は主に言語（日本語、韓国語、中国語、フランス語……）によって異なります。

日本語用キーボードの基本的な配列は決まりがあるのですが、パソコンの製品によって若干異なる場合があります。
この教材では基本的な JIS配列（ジス配列）と呼ばれる配列を前提として解説を進めていきます。

パソコンによって少し異なりますが、日本語キーボードのMacBook（MacBook, MacBook Air, MacBook Pro）の場合は以下のようなキー配列になっています。
また、最上部にタッチパネル（タッチバー）を搭載したモデルもあります。

MacBook の スペースキー

Shift（シフト）

タッチバー

MacBook Pro の一部のモデルでは、キーボードの一番上の段に横長のタッチパネル画面が搭載されているものがあります。
これを「タッチバー」といいます。

タッチバーは状況によって表示内容が変わります。

実際に入力してみよう

文字入力を始める前の準備として、「ライブ変換」を OFF にしましょう。

ライブ変換とは macOS の機能の 1 つで、入力した文字を自動的に漢字などに変換してくれる、というものです。初期設定ではこのライブ変換は ON になっていますが、意図していない文字に勝手に変換されるというデメリットも大きいため、慣れないうちは OFF にすることを推奨します。

「ライブ変換」を OFF にするには、メニューバー右上に表示されている「A」（または「あ」または「ア」）のアイコンをクリックし、「ライブ変換」をクリックします。
チェックマークが外れたら OFF の状態です。

文字の消し方

文字を入力していると、間違えることもあります。ここでは、入力した文字の消し方を学びましょう。

Delete（デリート）というキーを使います。
これを押すと、 1 つ前（左側）の文字を消す事ができます。

また、fn キー と Delete（デリート）を同時に押すことで、1 つ後ろ（右側）の文字を消すことが出来ます。

Delete キーは Windows と Mac で機能が少し違うので、両方の OS を使った経験がある人は間違えないように気をつけましょう。

日本語を入力する

PC で日本語入力をする場合、「ひらがな入力」と「ローマ字入力」があります。
ひらがな入力はキーボードに描かれているひらがなを押して入力するもので、ローマ字入力はアルファベットを押してローマ字の形で入力するものです。

これから PC を習得する場合、ローマ字入力を強くおすすめします。
なぜなら、ローマ字入力の方が覚えるキーの数が少なくて済むからです。ひらがな入力をする場合、英語のアルファベット26文字に加え、46文字のひらがなキー、さらに記号を覚える必要があります。
ローマ字入力なら、アルファベット26文字と記号を覚えるだけで済みます。

また、ローマ字入力のほうが記号や数字を入力しやすいというメリットもあります。PC で学校の課題に取り組んだり、会社の仕事をしたり、プログラミングをする場合には、日本語だけではなく必ず記号や数字を入力することになります。
ひらがな入力の場合、切り替えの手間が増えてしまいます。

こういった理由から N 予備校でも基本的にはローマ字入力を使います。
もし、ローマ字入力に自信がない……という方は、付録の「ローマ字入力を覚えよう」のテキストをご覧ください。

ホームポジション

キーボードはどんな指でも押すことができますが、正しい入力姿勢が決まっています。これが「ホームポジション」です。
きちんとホームポジションで入力できるようになると、高速でミスの少ない入力が可能になります。また、キーに書かれている文字を見ずに、画面を見たまま入力する タッチタイピング も可能になります。

左手の人指し指を F、右画の人差し指を J に置いた状態がホームポジションになります。
以下の画像を参考に、正しい指で入力してみてください。

ホームポジション

一度、変な入力の癖がついてしまうと正しい入力姿勢を習得するのは大変です。出来る限り早めにホームポジションを習得するのが望ましいでしょう。
ただし、「ホームポジションで入力出来ないとパソコンを使ってはいけない」というわけではありません。気楽に少しずつキーボードに慣れていきましょう。
